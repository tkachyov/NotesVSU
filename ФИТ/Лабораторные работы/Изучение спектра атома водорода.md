
### 1. Цель работы
1. Исследование спектра испускания атомарного водорода.
2. Наблюдение линейчатого спектра водорода с помощью монохроматора.
3. Измерение длин волн спектральных линий и их идентификация (серия Бальмера).

---

### 2. Устройство и принцип работы установки

#### 2.1. Состав установки
Установка состоит из двух основных частей:
1. **Объект исследования (излучатель)** — водородная лампа с блоком питания и юстировочным узлом.
2. **Измерительное устройство** — монохроматор типа МУМ (спектрометр).

#### 2.2. Излучатель
- Внутри корпуса находятся водородная лампа и блок высокого напряжения.
- На корпусе расположены: выходное окно с блендой, выключатель сети, индикатор включения, винты юстировки для настройки направления излучения.
- Излучатель подключается к сети 220 В, 50 Гц.

#### 2.3. Монохроматор (спектрометр)
Монохроматор служит для разложения излучения в спектр и измерения длин волн. Основные элементы (см. рис. 2 в методичке):

1. **Входная щель** (1) — с микрометрическим винтом (9) для регулировки ширины (обычно 0,02–0,03 мм).
2. **Коллиматорный объектив** (2) — формирует параллельный пучок света, фокусируется винтом (8).
3. **Сложная призма** (3) — состоит из трёх склеенных призм (две из тяжёлого флинта, одна из крона). Обеспечивает высокую дисперсию (разложение света).
4. **Поворотный столик** (6) с микрометрическим винтом (7) и отсчётным барабаном — позволяет поворачивать призму и выводить в поле зрения разные участки спектра.
5. **Зрительная труба** (объектив 4, окуляр 5) — внутри находится указатель (10), по которому совмещают спектральную линию для отсчёта.
6. **Оптическая скамья** — для размещения источника света и конденсора.
7. **Пульт управления** — для включения ламп и подсветки шкалы.

> В данной работе выходная щель не используется; наблюдение ведётся визуально через окуляр.

### 3. Теоретическое введение

#### 3.1. Квантование энергии в атоме водорода
Атом водорода может находиться только в определённых энергетических состояниях. При переходе электрона с уровня $m$ (с более высокой энергией) на уровень $n$ (с более низкой) излучается квант света с энергией:

$$
E = h \nu = \frac{hc}{\lambda} = E_m - E_n
$$

где $h$ — постоянная Планка, $\nu$ — частота, $\lambda$ — длина волны.

#### 3.2. Энергетические уровни
Энергия уровней водорода определяется главным квантовым числом $n$:

$$
E_n \propto -\frac{1}{n^2}
$$

#### 3.3. Формула для длин волн (обобщённая формула Бальмера)
Длины волн спектральных линий водорода описываются формулой:

$$
\lambda_{mn} = R \left( \frac{1}{n^2} - \frac{1}{m^2} \right)
$$

или в волновых числах:

$$
\frac{1}{\lambda} = R \left( \frac{1}{n^2} - \frac{1}{m^2} \right)
$$

где:
- $R = 109 \text{ } 677{,}6\ \text{см}^{-1}$ — постоянная Ридберга,
- $n$ — номер нижнего уровня,
- $m$ — номер верхнего уровня $( m > n)$.

#### 3.4. Серия Бальмера
В видимой области спектра находится **серия Бальмера**, для которой $n = 2$.

Первые четыре линии серии Бальмера:

| Обозначение | $m$ | Цвет         |
| ----------- | --- | ------------ |
| $H_\alpha$  | 3   | красный      |
| $H_\beta$   | 4   | сине-зелёный |
| $H_\gamma$  | 5   | синий        |
| $H_\delta$  | 6   | фиолетовый   |

### 4. Подготовка и проведение измерений

#### 4.1. Юстировка и настройка
1. Установить излучатель на оптической скамье на расстоянии 40–50 см от входной щели монохроматора.
2. Установить конденсор между источником и щелью (13–15 см от источника) для концентрации света.
3. Включить водородную лампу.
4. Отрегулировать юстировочными винтами направление излучения на щель.
5. Установить нужную ширину входной щели (0,02–0,03 мм).

#### 4.2. Фокусировка монохроматора
1. Вращая окуляр (5), добиться резкого изображения указателя (10).
2. При наличии ртутной лампы (или по линиям водорода) вращать винт (8) коллиматорного объектива для получения чётких спектральных линий.

#### 4.3. Измерение длин волн
1. Вращая барабан (7), поворачивать призму и наблюдать спектральные линии.
2. Совместить центр линии с остриём указателя.
3. Снять отсчёт по барабану.
4. По градуировочному графику (или по известным линиям ртути) определить длину волны.
5. Для слабых линий можно слегка поворачивать барабан в обе стороны — глаз лучше замечает движение линии.

---

### 5. Обработка результатов (план)

1. **Измеренные длины волн** сравнить с теоретическими значениями для серии Бальмера.
2. Вычислить **постоянную Ридберга** по формуле:
   $$R = \frac{1}{\lambda \left( \frac{1}{2^2} - \frac{1}{m^2} \right)}$$
   для каждой линии.
3. Найти среднее значение $R$ и сравнить с табличным.
4. Оценить погрешности измерений.

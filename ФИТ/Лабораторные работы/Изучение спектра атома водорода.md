## 2. Устройство и принцип работы установки

### 2.1. Состав установки

Установка состоит из двух основных частей:  
1. **Объект исследования (излучатель)** — водородная лампа с блоком питания и юстировочным узлом.  
2. **Измерительное устройство** — монохроматор типа МУМ (спектрометр).

### 2.2. Водородная трубка

В качестве источника света используется водородная трубка Н-образной формы, питаемая от катушки Румкорфа. Наибольшая яркость спектра достигается, когда источником света служит торец горизонтальной части трубки (капилляра).  

Для увеличения яркости линий атомарного водорода в состав газа иногда добавляют пары воды. Молекулы воды в электрическом разряде разлагаются, образуя атомарный водород. Трубка заполняется газом до давления 5–10 Торр.

В спектре водородной трубки наряду с линиями атомного спектра наблюдается спектр молекулярного водорода. Поиск нужных линий следует начинать с наиболее интенсивной красной линии $H_\alpha$. Вторая линия, $H_\beta$, — зелено-голубая. В промежутке между $H_\alpha$ и $H_\beta$ располагаются несколько красно-жёлтых и зелёных сравнительно слабых молекулярных полос. Третья линия, $H_\gamma$, — фиолетово-синяя. Перед этой линией располагаются две слабые размазанные молекулярные полосы синего цвета. Четвёртая линия, $H_\delta$, — фиолетовая. Её не всегда удаётся найти.

### 2.3. Монохроматор (спектрометр)

Монохроматор служит для разложения излучения в спектр и измерения длин волн. Основные элементы:  

1. **Входная щель** — с микрометрическим винтом для регулировки ширины (обычно 0,02–0,03 мм).  
2. **Коллиматорный объектив** — формирует параллельный пучок света, фокусируется винтом.  
3. **Сложная призма** — состоит из трёх склеенных призм (две из тяжёлого флинта, одна из крона). Обеспечивает высокую дисперсию (разложение света).  
4. **Поворотный столик** с микрометрическим винтом и отсчётным барабаном — позволяет поворачивать призму и выводить в поле зрения разные участки спектра.  
5. **Зрительная труба** (объектив, окуляр) — внутри находится указатель, по которому совмещают спектральную линию для отсчёта.  
6. **Оптическая скамья** — для размещения источника света и конденсора.  
7. **Пульт управления** — для включения ламп и подсветки шкалы.

## 3. Теоретическое введение

### 3.1. Квантование энергии в атоме водорода

Атом водорода может находиться только в определённых энергетических состояниях. При переходе электрона с уровня $m$ (с более высокой энергией) на уровень $n$ (с более низкой) излучается квант света с энергией:

$$
E = h \nu = \frac{hc}{\lambda} = E_m - E_n
$$

где $h$ — постоянная Планка, $\nu$ — частота, $\lambda$ — длина волны.

### 3.2. Энергетические уровни

Энергия уровней водорода определяется главным квантовым числом $n$:

$$
E_n \propto -\frac{1}{n^2}
$$

### 3.3. Формула для длин волн (обобщённая формула Бальмера)

Длины волн спектральных линий водорода описываются формулой:

$$
\frac{1}{\lambda} = R \left( \frac{1}{n^2} - \frac{1}{m^2} \right)
$$

где:  
- $R = 109\,677{,}6\ \text{см}^{-1}$ — постоянная Ридберга,  
- $n$ — номер нижнего уровня,  
- $m$ — номер верхнего уровня ($m > n$).

### 3.4. Серия Бальмера

В видимой области спектра находится **серия Бальмера**, для которой $n = 2$.

Первые четыре линии серии Бальмера:

| Обозначение | $m$ | Цвет        |
|-------------|-----|-------------|
| $H_\alpha$  | 3   | красный     |
| $H_\beta$   | 4   | сине-зелёный|
| $H_\gamma$  | 5   | синий       |
| $H_\delta$  | 6   | фиолетовый  |

## 5. Подготовка и проведение измерений

### 5.1. Юстировка и настройка

1. Установить излучатель на оптической скамье на расстоянии 40–50 см от входной щели монохроматора.  
2. Установить конденсор между источником и щелью (13–15 см от источника) для концентрации света.  
3. Включить водородную лампу.  
4. Отрегулировать юстировочными винтами направление излучения на щель.  
5. Установить нужную ширину входной щели (0,02–0,03 мм).

### 5.2. Фокусировка монохроматора

1. Вращая окуляр, добиться резкого изображения указателя.  
2. При помощи ртутной лампы (или по линиям водорода) вращать винт коллиматорного объектива для получения чётких спектральных линий.

### 5.3. Градуировка спектрометра

Для градуировки используются спектры ртути и неона с известными длинами волн. Для каждой линии снимается отсчёт по барабану. По полученным данным строится градуировочный график (зависимость длины волны от отсчёта), который затем используется для определения длин волн водородных линий.

### 5.4. Измерение длин волн водородных линий

1. Вращая барабан, поворачивать призму и наблюдать спектральные линии водорода.  
2. Совместить центр линии с остриём указателя.  
3. Снять отсчёт по барабану.  
4. По градуировочному графику определить длину волны.  
5. Для слабых линий можно слегка поворачивать барабан в обе стороны — глаз лучше замечает движение линии.

## 6. Обработка результатов

### 6.1. Расчётные формулы

Постоянная Ридберга для каждой линии вычисляется по формуле:

$$
R = \frac{1}{\lambda \left( \frac{1}{2^2} - \frac{1}{m^2} \right)}
$$

Теоретическое значение постоянной Ридберга в системе СГС:

$$
R = 2\pi^2 \frac{m_e e^4}{h^3 c}
$$

где $e$ — заряд электрона, $m_e$ — масса электрона, $c$ — скорость света.

### 6.2. Таблица измерений

| Линия | $m$ | Отсчёт по барабану | $\lambda_{\text{эксп}}$, нм | $\lambda_{\text{теор}}$, нм | $R$, см$^{-1}$ |
|-------|-----|--------------------|----------------------------|----------------------------|----------------|
| $H_\alpha$ | 3 | | | 656,3 | |
| $H_\beta$ | 4 | | | 486,1 | |
| $H_\gamma$ | 5 | | | 434,0 | |
| $H_\delta$ | 6 | | | 410,2 | |

*Примечание: экспериментальные значения $\lambda$ определяются по градуировочному графику.*

### 6.3. Вычисление постоянной Ридберга

Среднее значение постоянной Ридберга:

$$
\overline{R} = \frac{R_1 + R_2 + R_3 + R_4}{4}
$$

### 6.4. Оценка погрешностей

Абсолютная погрешность для каждого измерения:

$$
\Delta R_i = |R_i - \overline{R}|
$$

Среднеквадратичная погрешность:

$$
\sigma_R = \sqrt{\frac{\sum_{i=1}^{4} (R_i - \overline{R})^2}{4(4-1)}}
$$

Относительная погрешность:

$$
\varepsilon = \frac{\sigma_R}{\overline{R}} \cdot 100\%
$$

## 8. Контрольные вопросы

### 1. Определите по рис. 1 потенциал ионизации и энергию возбуждения атома водорода.

**Потенциал ионизации** — это минимальная энергия, необходимая для удаления электрона из атома, т.е. переход с основного уровня ($n=1$) на уровень $n = \infty$. Энергия ионизации равна $E_{\text{ион}} = \frac{hcR}{1^2} = hcR$, а потенциал ионизации — $U_{\text{ион}} = \frac{hcR}{e}$ (в вольтах).  
**Энергия возбуждения** — это энергия, необходимая для перехода электрона с основного уровня ($n=1$) на более высокий уровень $n$, например на $n=2$ (первый потенциал возбуждения). Она вычисляется по формуле $\Delta E = hcR \left( \frac{1}{1^2} - \frac{1}{n^2} \right)$.

### 2. Что такое изотопический сдвиг?

Изотопический сдвиг — это смещение спектральных линий у атомов различных изотопов одного и того же элемента. Он возникает из-за различия в массах ядер (и, следовательно, в приведённых массах электрона) и из-за различия в радиусах ядер (объёмный эффект). Для водорода изотопический сдвиг между $^1\text{H}$ и $^2\text{H}$ (дейтерием) проявляется в небольшом различии длин волн соответствующих линий.

### 3. Получите выражение (4) из принципа соответствия Бора.

Выражение (4) — это формула для постоянной Ридберга:

$$
R = 2\pi^2 \frac{m_e e^4}{h^3 c}
$$

По принципу соответствия Бора, в пределе больших квантовых чисел ($n \gg 1$) частоты излучения квантового атома должны совпадать с частотами, предсказываемыми классической электродинамикой для излучения вращающегося электрона. Для круговых орбит в модели Бора момент импульса квантуется: $m_e v r = n \hbar$, где $\hbar = h/(2\pi)$. Частота вращения электрона на орбите $\nu = v/(2\pi r)$. Сравнивая квантовую формулу для частоты перехода между соседними уровнями $\nu = \frac{Rc}{(n+1)^2} - \frac{Rc}{n^2} \approx \frac{2Rc}{n^3}$ с классической частотой вращения электрона $\nu_{\text{кл}} = \frac{e}{2\pi r} \sqrt{\frac{e}{m_e r}}$, и используя квантование момента импульса, можно получить выражение для $R$, совпадающее с (4). Этот вывод сыграл ключевую роль в установлении связи между квантовой теорией и классической физикой.

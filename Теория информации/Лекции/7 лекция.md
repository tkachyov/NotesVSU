# Конспект лекции №7: Энтропия марковских источников, равномерное кодирование и закон больших чисел

## 1. Случайные блуждания на графах как пример марковского источника

### 1.1. Определение случайного блуждания
Рассматривается неориентированный граф $G(V, E)$ с множеством вершин $V = \{1, 2, \dots, \nu\}$ и матрицей смежности $M_{ij}$. Частица случайно переходит из вершины в вершину. Последовательность вершин $\{X_t\}$ образует случайное блуждание.

**Переходные вероятности** (вероятность перейти из $i$ в $j$):
$$\pi_{ij} = \frac{M_{ij}}{\sum_k M_{ik}} = \frac{M_{ij}}{M_i},$$
где $M_i = \sum_k M_{ik}$ — степень вершины $i$ (количество инцидентных рёбер).

### 1.2. Стационарное распределение
Обозначим:
- $|E| = \frac{1}{2} \sum_i M_i$ — общее количество рёбер в графе.
- **Стационарное распределение** (пропорционально степени вершины):
    $$p_i = \frac{M_i}{2|E|}.$$

**Проверка условия стационарности $p\Pi = p$:**
$$\sum_i p_i \pi_{ij} = \sum_i \frac{M_i}{2|E|} \cdot \frac{M_{ij}}{M_i} = \frac{1}{2|E|} \sum_i M_{ij} = \frac{1}{2|E|} \sum_i M_{ji} = \frac{M_j}{2|E|} = p_j.$$

### 1.3. Энтропия на сообщение для случайного блуждания
Поскольку это цепь Маркова связности $s=1$, энтропия на сообщение $H_\infty = H(X_2 | X_1)$:
$$H(X|X^\infty) = -\sum_i p_i \sum_j \pi_{ij} \log \pi_{ij} = \log(2|E|) - H\left(\frac{M_1}{2|E|}, \frac{M_2}{2|E|}, \dots, \frac{M_\nu}{2|E|}\right).$$

### 1.4. Примеры
**Пример 2.13 (граф с 5 вершинами и 8 рёбрами):**
- $|V|=5$, $|E|=8$, степени вершин: $(3,3,3,3,4)$.
- Стационарное распределение: $p = (3,3,3,3,4)/16$.
- Энтропия на сообщение: $H_\infty = \frac{3}{4}\log 3 + \frac{1}{2} \approx 1.6887$ бит.
- Взаимная информация между соседними состояниями: $I(X_{n+1}; X_n) = 3 - \frac{3}{2}\log 3 \approx 0.6225$ бит.

**Пример (гиперкуб $Q_d$):**
- $|V| = 2^d$, $|E| = d \cdot 2^{d-1}$.
- Стационарное распределение равномерно: $p = (2^{-d}, \dots, 2^{-d})$.
- Энтропия на сообщение: $H_\infty = \log d$ бит.
- Взаимная информация: $I = d - \log d$ бит.

## 2. Равномерное кодирование (Fixed-to-Fixed, FF)

### 2.1. Основные понятия
- **Блок источника:** $N$ символов источника из алфавита $X$.
- **Кодовое слово:** $n$ двоичных символов.
- **Скорость кода:**
    $$R = \frac{\lceil \log |C| \rceil}{N} \text{ бит/символ источника}.$$
- Для кода $C = \{0,1\}^n$ (все двоичные последовательности длины $n$):
    $$R = \frac{n}{N} \text{ бит/символ}.$$

### 2.2. Условие однозначного кодирования
Если отображение $X^N \to C$ взаимно однозначно, то необходимо $|X|^N \le |C|$, откуда:
$$R \ge \log |X| \ge H(X).$$
Для неравновероятных букв $H(X) < \log |X|$, поэтому равномерный код всегда будет иметь скорость **больше** энтропии источника.

### 2.3. Кодирование с ошибками
Если $|X|^N > |C|$, то невозможно закодировать все последовательности однозначно.
- Выбирается **множество однозначно кодируемых последовательностей** $T \subset X^N$ с $|T| = |C|$.
- Остальным последовательностям сопоставляются произвольные кодовые слова (например, все одно и то же).
- **Вероятность ошибки:**
    $$P_e = P\{x \notin T\} = 1 - P(T).$$

### 2.4. Пример 2.14 (троичный источник)
- $X = \{a,b,c\}$, $p(a)=1/2$, $p(b)=1/3$, $p(c)=1/6$.
- $N=2$, $X^N$ содержит 9 пар.
- Используется код $C = \{0,1\}^3$ ($|C|=8$).
- Скорость: $R = 3/2 = 1.5$ бит/символ.
- Оптимальный выбор $T$ — все пары, кроме $cc$. Вероятность ошибки: $P_e = p(cc) = 1/36 \approx 0.0278$.

## 3. Постановка задачи кодирования источника

### 3.1. Прямая и обратная теоремы кодирования
Число $H$ называется **скоростью создания информации** источником, если:

1.  **Прямая теорема:** Для любого $R > H$ существует равномерный код со скоростью $R$, обеспечивающий сколь угодно малую вероятность ошибки ($P_e \to 0$).
2.  **Обратная теорема:** Для любого $R < H$ вероятность ошибки не может быть сделана меньше некоторой положительной константы (т.е. $P_e$ не стремится к 0).

### 3.2. Скорость создания информации
Для стационарных источников скорость создания информации совпадает с энтропией на сообщение:
$$H = H(X|X^\infty) = H_\infty(X).$$

### 3.3. Типы кодов
- **FF (Fixed-to-Fixed):** блок фиксированной длины → слово фиксированной длины.
- **VF (Variable-to-Fixed):** блок переменной длины → слово фиксированной длины.
- **FV (Fixed-to-Variable):** блок фиксированной длины → слово переменной длины.
- **VV (Variable-to-Variable):** блок переменной длины → слово переменной длины.

Далее рассматриваются FF-коды для дискретного постоянного источника (без памяти).

## 4. Неравенство Чебышёва и закон больших чисел

### 4.1. Неравенство Маркова (для неотрицательных величин)
Для случайной величины $x \ge 0$ и $A > 0$:
$$P(x \ge A) \le \frac{M[x]}{A}.$$

### 4.2. Неравенство Чебышёва
Для произвольной случайной величины $x$ с математическим ожиданием $m_x$ и дисперсией $\sigma_x^2 = D[x]$:
$$P(|x - m_x| \ge \varepsilon) \le \frac{\sigma_x^2}{\varepsilon^2}.$$

**Доказательство:** применяется неравенство Маркова к $y = (x - m_x)^2$.

### 4.3. Закон больших чисел
Пусть $X_1, \dots, X_n$ — независимые одинаково распределённые случайные величины с $M[X_i] = m$ и $D[X_i] = \sigma^2$. Тогда для любого $\varepsilon > 0$:
$$P\left(\left|\frac{1}{n}\sum_{i=1}^n X_i - m\right| \ge \varepsilon\right) \le \frac{\sigma^2}{n\varepsilon^2}.$$
**Следствие:** при $n \to \infty$ среднее арифметическое сходится по вероятности к $m$:
$$\lim_{n \to \infty} P\left(\left|\frac{1}{n}\sum_{i=1}^n X_i - m\right| \ge \varepsilon\right) = 0.$$

### 4.4. Эргодические процессы
**Определение:** Стационарный процесс называется **эргодическим**, если для него справедлив закон больших чисел (среднее по времени равно среднему по множеству реализаций).
- Примеры: последовательность независимых одинаково распределённых величин, неприводимая непериодическая цепь Маркова.

## 5. Прямая теорема кодирования для дискретного постоянного источника

### 5.1. Упрощающие предположения
- Источник без памяти (дискретный постоянный источник).
- Алфавит конечен, все вероятности букв положительны.
- Собственная информация имеет конечную энтропию $H$ и конечную дисперсию.

### 5.2. Формулировка теоремы
**Теорема 2.7:** Пусть $H$ — энтропия дискретного постоянного источника. Для любых $\varepsilon, \delta > 0$ существует $n_0$ такое, что для любого $n > n_0$ найдётся равномерный код, кодирующий источник блоками длины $n$ со скоростью $R \le H + \delta$ и вероятностью ошибки $P_e \le \varepsilon$.

### 5.3. Идея доказательства (намечена в лекции)
Используется закон больших чисел:
- Для достаточно длинных блоков $x_1 \dots x_n$ эмпирическая энтропия $-\frac{1}{n}\log p(x_1 \dots x_n)$ близка к $H$.
- Множество "типичных" последовательностей имеет вероятность, близкую к 1, и его размер примерно $2^{nH}$.
- Поскольку $R = n/N$ и $R > H$, можно выбрать $T$ как множество типичных последовательностей (или его подмножество), тогда $|T| \approx 2^{nH} \le 2^{nR} = |C|$.
- Остальные последовательности (нетипичные) имеют малую суммарную вероятность $\le \varepsilon$.

### 5.4. Смысл теоремы
- При скорости $R$, **превышающей** энтропию $H$, можно сделать вероятность ошибки сколь угодно малой.
- В следующей лекции будет доказана обратная теорема: при $R < H$ ошибка неизбежна.

## Итоги лекции
1.  **Случайные блуждания на графах** дают наглядный пример марковского источника с простым выражением для энтропии.
2.  **Равномерное кодирование** (FF) с фиксированной скоростью $R$ неизбежно приводит к ошибкам, если $R$ меньше $\log|X|$, но можно выбирать множество однозначно кодируемых последовательностей $T$, чтобы минимизировать $P_e$.
3.  **Скорость создания информации** источником равна $H_\infty$ — энтропии на сообщение.
4.  **Закон больших чисел** (следствие неравенства Чебышёва) утверждает, что среднее арифметическое независимых величин сходится к их математическому ожиданию.
5.  **Прямая теорема кодирования** гарантирует, что для дискретного постоянного источника при скорости $R > H$ можно добиться сколь угодно малой вероятности ошибки.

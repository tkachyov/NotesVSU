# Конспект лекции №3: Измерение информации. Собственная информация и энтропия

## 1. Собственная информация

### 1.1. Мотивация и требования к мере информации
Задача количественного измерения информации возникла из практической потребности минимизировать затраты (количество сигналов) на передачу сообщений. Поскольку сообщения рассматриваются как случайные события, мера информации должна удовлетворять следующим интуитивным требованиям:

1.  **Неотрицательность:** Мера должна быть неотрицательной, так как она отражает затраты.
2.  **Зависимость от вероятности:** Мера должна определяться исключительно вероятностью сообщения: $\mu(x) = \mu(p(x))$.
3.  **Монотонность для равновероятных событий:** Для кодирования $2^m$ равновероятных сообщений требуется $m$ двоичных символов. Следовательно, мера должна удовлетворять соотношению:
    $$\mu\left(\frac{1}{2^m}\right) = m \cdot \mu\left(\frac{1}{2}\right).$$
    Это свойство наталкивает на логарифмическую зависимость.
4.  **Аддитивность для независимых сообщений:** Если сообщения $x_1, \dots, x_n$ статистически независимы, то затраты на передачу всей последовательности должны равняться сумме затрат на передачу каждого сообщения в отдельности:
    $$\mu(x_1, \dots, x_n) = \mu(x_1) + \dots + \mu(x_n).$$

### 1.2. Определение собственной информации
Из перечисленных требований (особенно аддитивности для независимых событий и связи с вероятностью) следует логарифмическая мера.

**Определение:** Собственной информацией $I(x)$ сообщения $x$, выбираемого из дискретного ансамбля $\mathbf{X} = \{x, p(x)\}$, называется величина:
$$I(x) = -\log p(x).$$

### 1.3. Единицы измерения информации
Выбор основания логарифма определяет единицу измерения:
- **Основание 2:** **бит** (binary digit). Наиболее удобен для практики, так как сразу показывает требуемое количество двоичных символов.
- **Натуральное основание ($e$):** **нат** (natural unit). Удобен в теоретических выкладках из-за простоты дифференцирования $\ln x$.
- **Основание 10:** **хартли** (dit). Использовался Хартли до Шеннона.

**Соотношения:** $1 \text{ бит} = \ln 2 \approx 0.693 \text{ нат}$.

### 1.4. Свойства собственной информации
1.  **Неотрицательность:** $I(x) \ge 0$ для всех $x \in X$.
2.  **Монотонность:** Чем вероятнее событие, тем меньше информации оно несёт.
    $$p(x_1) > p(x_2) \Rightarrow I(x_1) < I(x_2).$$
3.  **Аддитивность для независимых событий:** Если $x_1, \dots, x_n$ независимы, то информация в последовательности равна сумме информаций отдельных сообщений:
    $$I(x_1, \dots, x_n) = \sum_{i=1}^{n} I(x_i).$$

## 2. Энтропия дискретного ансамбля

Собственная информация — это случайная величина. Для характеристики ансамбля в целом используется её среднее значение.

### 2.1. Определение
**Энтропией** $H(X)$ дискретного вероятностного ансамбля $\mathbf{X} = \{x, p(x)\}$ называется математическое ожидание собственной информации:
$$H(X) = \mathbf{M}[-\log p(x)] = -\sum_{x \in X} p(x) \log p(x).$$

**Интерпретация:**
- Мера **априорной неосведомлённости** о том, какое именно сообщение будет порождено источником.
- Мера **неопределённости** ансамбля.
- Среднее количество информации, приходящееся на один символ источника.
- Нижняя граница среднего количества бит, необходимого для представления (сжатия) одного символа.

### 2.2. Основные свойства энтропии
1.  **Неотрицательность:** $H(X) \ge 0$. Равенство нулю достигается, когда один из символов имеет вероятность 1 (полная определённость), а остальные — 0.
2.  **Верхняя граница:** Энтропия максимальна, когда все элементы ансамбля равновероятны.
    $$H(X) \le \log |X|.$$
    Равенство достигается тогда и только тогда, когда $p(x) = \frac{1}{|X|}$ для всех $x \in X$.
    *Пример:* Для алфавита из 256 символов ($|X|=256$) максимальная энтропия равна $\log_2 256 = 8$ бит/символ.
3.  **Инвариантность к перестановке:** Энтропия зависит только от распределения вероятностей, а не от порядка следования символов. Если распределения вероятностей для $X$ и $Y$ представляют собой одинаковые наборы чисел (в разном порядке), то $H(X) = H(Y)$.
4.  **Аддитивность для независимых ансамблей:** Если $\mathbf{X}$ и $\mathbf{Y}$ независимы, то энтропия совместного ансамбля равна сумме энтропий:
    $$H(XY) = H(X) + H(Y).$$
5.  **Выпуклость вверх:** Энтропия является строго **выпуклой вверх (вогнутой)** функцией от распределения вероятностей.
6.  **Выравнивание вероятностей:** Любая операция, которая "выравнивает" вероятности элементов ансамбля (делает распределение более равномерным), увеличивает энтропию.
7.  **Обработка не увеличивает информацию:** Если $Y = g(X)$ — результат некоторой (детерминированной) обработки ансамбля $X$, то энтропия результата не превосходит энтропии исходного ансамбля:
    $$H(Y) \le H(X).$$
    Равенство достигается только в том случае, если отображение $g$ является обратимым (биективным). *Смысл:* Обработка информации (например, сортировка или округление) не добавляет новой информации, а чаще всего уничтожает часть неопределённости.

## 3. Пример применения свойств (Русский текст)
- Если для русского текста использовать частоты букв, энтропия составит около 4.5 бит на букву.
- Это означает, что в среднем для хранения буквы достаточно 4.5 бит, хотя стандартный байт (8 бит) используется неэффективно (архиваторы сжимают текст примерно в 1.8 раза, а не в 4, из-за наличия памяти в тексте, то есть зависимости между буквами).

## 4. Двоичный ансамбль (Бернуллиевский источник)

Рассмотрим важнейший частный случай: ансамбль $X = \{0, 1\}$.
- Пусть $p(1) = p$, $p(0) = 1 - p = q$.

**Энтропия двоичного ансамбля** обозначается специальным символом:
$$H(X) = -p \log p - (1-p) \log (1-p) \triangleq \eta(p).$$

### 4.1. Анализ функции $\eta(p)$
1.  **Граничные точки:**
    - При $p=0$: $\eta(0) = 0$.
    - При $p=1$: $\eta(1) = 0$.
    *Доказательство через правило Лопиталя:* $\lim_{p\to 0} p\log p = 0$.
2.  **Симметрия:** $\eta(p) = \eta(1-p)$ (свойство 3 энтропии).
3.  **Экстремум:** Найдём производную и приравняем к нулю.
    $$\frac{d\eta}{dp} = -\log p + \log (1-p).$$
    Равенство нулю достигается при $p = 1-p$, т.е. $p = \frac{1}{2}$.
    Вторая производная:
    $$\frac{d^2\eta}{dp^2} = -\frac{\log e}{p(1-p)} < 0.$$
    Это подтверждает, что точка $p=\frac{1}{2}$ — точка **максимума**.
    $$H_{\max} = \eta\left(\frac{1}{2}\right) = \log_2 2 = 1 \text{ бит}.$$

**Вывод:** Один "инженерный бит" (двоичный разряд) может нести максимум 1 бит информации по Шеннону. Если вероятности неравны, информационная нагрузка на символ падает (например, при $p=0.1$, $H \approx 0.47$ бит).

### 4.2. График функции $\eta(p)$
Функция имеет колоколообразную форму, симметричную относительно $p=0.5$, достигая максимума в 1 бит и спадая до нуля на краях интервала.

## 5. Термодинамическая энтропия (Связь с физикой)

В лекции проводится параллель между энтропией Шеннона и энтропией Больцмана в статистической физике.

1.  **Статистический вес $W$:** Количество микросостояний системы, реализующих данное макросостояние. В равновесии все микросостояния равновероятны.
2.  **Формула Больцмана:** Для обеспечения аддитивности (т.к. $W$ мультипликативен) вводится энтропия:
    $$S = k \ln W,$$
    где $k$ — постоянная Больцмана.
3.  **Расчёт для идеального газа:**
    $$W = \frac{N!}{n_1! n_2! \dots n_m!},$$
    где $n_i$ — число частиц в $i$-м состоянии, $\sum n_i = N$, $p_i = n_i/N$.
    Используя формулу Стирлинга ($\ln N! \approx N \ln N - N$), получаем:
    $$S = k \ln W \approx -kN \sum_{i=1}^m p_i \ln p_i.$$

**Итог:** Энтропия Шеннона $H(X)$ — это, по сути, энтропия Больцмана, отнесённая к одной частице (с точностью до постоянного множителя $k$). Эта аналогия подчёркивает универсальность понятия энтропии как меры неопределённости или беспорядка, хотя в приложениях теории информации она играет самостоятельную роль.

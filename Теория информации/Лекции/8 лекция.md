# Конспект лекции №8: Прямая и обратная теоремы кодирования. Множество типичных последовательностей

## 1. Прямая теорема кодирования для дискретного постоянного источника

### 1.1. Упрощающие предположения
- Рассматривается **дискретный постоянный источник** (стационарный источник без памяти).
- Алфавит источника конечен, все вероятности букв положительны.
- Это гарантирует, что собственная информация имеет конечную энтропию $H$ и конечную дисперсию.

### 1.2. Формулировка прямой теоремы
**Теорема 2.7:** Пусть $H$ — энтропия дискретного постоянного источника. Для любых $\varepsilon, \delta > 0$ существует $n_0$ такое, что для любого $n > n_0$ найдётся равномерный код, кодирующий источник блоками длины $n$ со скоростью $R \le H + \delta$ и вероятностью ошибки $P_e \le \varepsilon$.

*Смысл:* При скорости, превышающей энтропию, можно сделать вероятность ошибки сколь угодно малой.

### 1.3. Ключевая идея доказательства
Вводится **множество типичных последовательностей**:
$$T = \left\{ x \in X^n : \left| \frac{1}{n} l(x) - H \right| \le \delta_0 \right\},$$
где $l(x) = -\log p(x)$ — собственная информация последовательности.

Свойства множества $T$ (вытекают из закона больших чисел):
1.  $P(T) \ge 1 - \varepsilon$ (типичные последовательности имеют вероятность, близкую к 1).
2.  $|T| \le 2^{n(H + \delta_0)}$ (верхняя оценка мощности).
3.  Для всех $x \in T$: $p(x) \ge 2^{-n(H + \delta_0)}$ (нижняя оценка вероятностей).

### 1.4. Выбор параметров
- Положим $\delta_0 = \delta/2$.
- Из неравенства Чебышёва: при $n \ge n_{01} = D[I(x)]/(\delta^2 \varepsilon)$ имеем $P(T) \ge 1 - \varepsilon$, откуда $P_e \le \varepsilon$.
- При $n \ge n_{02} = 2/\delta$ выполняется $R < H + \delta$.
- Выбираем $n_0 = \max(n_{01}, n_{02})$.

**Вывод:** Кодируя только типичные последовательности (а их примерно $2^{nH}$), можно достичь скорости, сколь угодно близкой к $H$, с малой вероятностью ошибки.

## 2. Обратная теорема кодирования

### 2.1. Мотивация
Обратная теорема показывает, что при скорости **меньшей**, чем энтропия, невозможно добиться сколь угодно малой вероятности ошибки. Это спасает инженеров от бесплодных попыток "сжать" информацию сильнее теоретического предела.

### 2.2. Формулировка обратной теоремы
**Теорема 2.8:** Для дискретного постоянного источника с энтропией $H$ существует $\varepsilon > 0$ такое, что для любого $\delta > 0$ и для любого равномерного кода со скоростью $R \le H - \delta$ вероятность ошибки удовлетворяет неравенству $P_e \ge \varepsilon$.

*Смысл:* Если скорость хотя бы на сколь угодно малую $\delta$ меньше $H$, то вероятность ошибки не может быть сделана произвольно малой — она ограничена снизу положительной константой.

### 2.3. Доказательство
Код задаётся множеством однозначно кодируемых последовательностей $T_1$, причём $|T_1| \le 2^{nR} \le 2^{n(H-\delta)}$.

Вероятность правильного декодирования:
$$P_c = 1 - P_e = \sum_{x \in T_1} p(x).$$

Разбиваем сумму на две части:
$$P_c = \sum_{x \in T_1 \cap T} p(x) + \sum_{x \in T_1 \cap T^c} p(x),$$
где $T$ — множество типичных последовательностей (с параметром $\delta_0 = \delta/2$).

**Оценка первой суммы:**
$$\sum_{x \in T_1 \cap T} p(x) \le |T_1| \cdot \max_{x \in T} p(x) \le 2^{n(H-\delta)} \cdot 2^{-n(H-\delta_0)} = 2^{-n(\delta - \delta_0)} = 2^{-n\delta/2}.$$

**Оценка второй суммы:**
$$\sum_{x \in T_1 \cap T^c} p(x) \le P(T^c) \le \frac{D[I(x)]}{n \delta_0^2} = \frac{4D[I(x)]}{n \delta^2}.$$

**Итоговая оценка сверху для $P_c$:**
$$P_c \le 2^{-n\delta/2} + \frac{4D[I(x)]}{n \delta^2}.$$

Правая часть стремится к $0$ при $n \to \infty$, следовательно $P_c \to 0$, а $P_e \to 1$.

### 2.4. Положительность ошибки при конечных $n$
- Для нулевой ошибки нужно $|T_1| \ge |X|^n$, что требует $R \ge \log|X| \ge H$.
- Это противоречит условию $R \le H - \delta$.
- Следовательно, $P_e > 0$ для всех $n$.
- Выбирая $\varepsilon = \min\{\varepsilon_0, 1/2\}$, где $\varepsilon_0$ — минимальная ошибка для $n < n_0$, получаем $P_e \ge \varepsilon$ для всех $n$.

## 3. Множество типичных последовательностей (подробно)

### 3.1. Определение
Для дискретного постоянного источника:
$$T_n(\delta) = \left\{ x \in X^n : \left| \frac{1}{n} \log \frac{1}{p(x)} - H(X) \right| \le \delta \right\}.$$

### 3.2. Основные свойства (теорема о типичных последовательностях)
1.  **Асимптотическая полнота:** $P(T_n(\delta)) \to 1$ при $n \to \infty$.
2.  **Мощность:** $|T_n(\delta)| \le 2^{n(H(X)+\delta)}$.
3.  **Нижняя оценка мощности:** $|T_n(\delta)| \ge (1-\varepsilon) 2^{n(H(X)-\delta)}$ для достаточно больших $n$.
4.  **Асимптотическая равновероятность:** Для всех $x \in T_n(\delta)$:
    $$2^{-n(H(X)+\delta)} \le p(x) \le 2^{-n(H(X)-\delta)}.$$

### 3.3. Композиционная интерпретация
Для последовательности $x$ обозначим $\tau_x(x)$ — число появлений символа $x$. Тогда:
$$p(x) = \prod_{x \in X} p(x)^{\tau_x(x)}.$$
Типичные последовательности — это те, в которых доля каждого символа $\tau_x(x)/n$ близка к $p(x)$.

### 3.4. Пример 2.15 (двоичный источник)
- $X = \{0,1\}$, $p(1)=p < 1/2$.
- $p(x) = p^t (1-p)^{n-t}$, где $t$ — число единиц.
- Условие типичности:
    $$\left| \frac{t}{n} - p \right| \le \tilde{\delta}, \quad \tilde{\delta} = \delta \left(\log \frac{1-p}{p}\right)^{-1}.$$
- При $p=0.1$, $H \approx 0.47$ бит, доля типичных последовательностей среди всех $2^n$ экспоненциально мала, но их суммарная вероятность близка к 1.

## 4. Обобщение на источники с памятью

### 4.1. Интуитивное обобщение
Для стационарного источника с памятью скорость создания информации равна $H_\infty = H(X|X^\infty)$.
Можно ожидать, что для "типичных" последовательностей:
$$p(x_1, \dots, x_n) \approx 2^{-n H_\infty},$$
а количество таких последовательностей $\approx 2^{n H_\infty}$.

### 4.2. Использование эргодичности
- Ключевой инструмент — **закон больших чисел** (сходимость среднего арифметического к математическому ожиданию).
- Для эргодических процессов (включая неприводимые непериодические цепи Маркова) закон больших чисел выполняется.
- Достаточно заменить оценку через дисперсию (неравенство Чебышёва) на более общие эргодические теоремы.

### 4.3. Подход через условные вероятности
Для источника с памятью можно определить типичные последовательности через условные энтропии:
$$\frac{1}{n} \left( l(x_1, \dots, x_m) + \sum_{i=m+1}^n l(x_i | x_{i-m}, \dots, x_{i-1}) \right) \approx H(X|X^m).$$
При $m \to \infty$ это приближается к $H_\infty$.

### 4.4. Неэргодические источники
- Неэргодический источник можно представить как смесь эргодических компонент (например, источник с переключателем между двумя режимами).
- Наблюдая одну реализацию, нельзя определить, с какой компонентой имеешь дело.
- Прямая и обратная теоремы требуют более аккуратного рассмотрения, но в итоге $H_\infty$ остаётся предельной скоростью сжатия.

## Итоги лекции

1.  **Прямая теорема кодирования:** При $R > H$ можно добиться $P_e \to 0$ (кодируем только типичные последовательности).
2.  **Обратная теорема кодирования:** При $R < H$ вероятность ошибки не стремится к нулю, а ограничена снизу положительной константой.
3.  **Множество типичных последовательностей** — центральное понятие: почти все вероятностная масса сосредоточена на примерно $2^{nH}$ последовательностях, каждая из которых имеет вероятность $\approx 2^{-nH}$.
4.  **Эргодичность** — достаточное условие для распространения теорем на источники с памятью; скорость создания информации для стационарного источника равна $H_\infty = H(X|X^\infty)$.

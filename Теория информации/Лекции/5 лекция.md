# Конспект лекции №5: Относительная энтропия, взаимная информация и теорема о переработке

## 1. Относительная энтропия (Расстояние Кульбака-Лейблера)

### 1.1. Определение
Относительная энтропия — это несимметричная мера "удалённости" двух вероятностных распределений. Она интерпретируется как величина потерь информации при замене истинного распределения $\{p(x)\}$ на предполагаемое (модельное) распределение $\{q(x)\}$.

**Определение:**
$$D(p \parallel q) = \sum_{x \in X} p(x) \log \frac{p(x)}{q(x)} = M_p\left[ \log \frac{p(x)}{q(x)} \right].$$

### 1.2. Свойства
1.  **Неотрицательность:** $D(p \parallel q) \ge 0$.
2.  **Тождество:** $D(p \parallel q) = 0$ тогда и только тогда, когда $p(x) = q(x)$ для всех $x \in X$.
3.  **Несимметричность:** В общем случае $D(p \parallel q) \neq D(q \parallel p)$, поэтому это не метрика (не удовлетворяет неравенству треугольника).

*Доказательство неотрицательности* использует неравенство Йенсена для выпуклой вверх функции $\log$:
$$-D(p\|q) = \sum p \log \frac{q}{p} \le \log \sum p \frac{q}{p} = \log \sum q \le \log 1 = 0.$$

### 1.3. Пример 2.7 (Двоичные распределения)
Для множества $X = \{0, 1\}$:
- $p(0)=1-r$, $p(1)=r$;
- $q(0)=1-s$, $q(1)=s$.

Тогда:
$$D(P \| Q) = (1-r) \log \frac{1-r}{1-s} + r \log \frac{r}{s}.$$
Аналогично для $D(Q \| P)$.

*Численный пример:* $r=1/2$, $s=1/4$.
$$D(P\|Q) = \frac{1}{2}\log\frac{1/2}{3/4} + \frac{1}{2}\log\frac{1/2}{1/4} = 1 - \frac{1}{2}\log 3 = 0.2075 \text{ бит}.$$
$$D(Q\|P) = \frac{3}{4}\log\frac{3/4}{1/2} + \frac{1}{4}\log\frac{1/4}{1/2} = \frac{3}{4}\log 3 - 1 = 0.1887 \text{ бит}.$$
Наглядно видна несимметричность.

## 2. Взаимная информация

### 2.1. Мотивация
Для пары ансамблей $X$ (вход канала) и $Y$ (выход канала) необходимо количественно измерить информацию об элементах $x \in X$, содержащуюся в наблюдаемых символах $y \in Y$.

### 2.2. Определение взаимной информации для событий
Взаимная информация между конкретными реализациями $x$ и $y$:
$$I(x; y) = I(x) - I(x|y) = \log \frac{p(x|y)}{p(x)}.$$
Это изменение информации о $x$ благодаря получению $y$. *Примечание:* точка с запятой отличает эту величину от собственной информации пары $I(x, y)$.

### 2.3. Пример 2.8 (Двоичный симметричный канал, ДСК)
Для ДСК с вероятностью ошибки $p$ и равновероятным входом ($p_X(0)=p_X(1)=1/2$):
- Совместные вероятности: $p(0,0)=p(1,1)=(1-p)/2$, $p(0,1)=p(1,0)=p/2$.
- Выходы также равновероятны.
- Условные вероятности: $p(0|0)=p(1|1)=1-p$, $p(0|1)=p(1|0)=p$.
- Взаимная информация для конкретных пар:
    - $I(0;0) = I(1;1) = \log(2(1-p))$,
    - $I(0;1) = I(1;0) = \log(2p)$.

*Крайние случаи:*
- При $p=0$ (идеальный канал): $I(0;0)=\log 2 = 1$ бит.
- При $p=1/2$ (канал полностью зашумлен): $I(0;1)=\log(1) = 0$.

## 3. Средняя взаимная информация

### 3.1. Определение
Усреднение $I(x;y)$ по всем возможным парам:
$$I(X; Y) = M[I(x; y)] = \sum_{x \in X} \sum_{y \in Y} p(x, y) \log \frac{p(x, y)}{p(x)p(y)} = D(p(x,y) \parallel p(x)p(y)).$$
Альтернативная форма:
$$I(X; Y) = \sum_{x, y} p(x, y) \log \frac{p(y|x)}{p(y)}.$$

### 3.2. Свойства средней взаимной информации
1.  **Симметричность:** $I(X; Y) = I(Y; X)$.
2.  **Неотрицательность:** $I(X; Y) \ge 0$.
3.  **Критерий независимости:** $I(X; Y) = 0$ тогда и только тогда, когда $X$ и $Y$ независимы.
4.  **Связь с энтропией:**
    $$I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(XY).$$
    *Интерпретация:* Взаимная информация — это уменьшение неопределённости одного ансамбля за счёт знания другого.
5.  **Верхняя граница:** $I(X; Y) \le \min\{H(X), H(Y)\}$.
6.  **Другая верхняя граница:** $I(X; Y) \le \min\{\log |X|, \log |Y|\}$.
7.  **Выпуклость (по $p(x)$):** $I(X;Y)$ — **выпуклая вверх ($\cap$)** функция от входного распределения $p(x)$ (при фиксированных переходных вероятностях канала). Это гарантирует существование максимума (пропускной способности канала).
8.  **Выпуклость (по $p(y|x)$):** $I(X;Y)$ — **выпуклая вниз ($\cup$)** функция от условных распределений (переходных вероятностей канала) при фиксированном входном распределении.

## 4. Условная взаимная информация

### 4.1. Определения
Для трёх ансамблей $X, Y, Z$:
- **Условная взаимная информация при фиксированном $z$:**
    $$I(X; Y | z) = \sum_{x, y} p(x, y|z) \log \frac{p(y|x, z)}{p(y|z)}.$$
- **Средняя условная взаимная информация:**
    $$I(X; Y | Z) = M[I(X; Y | z)] = \sum_{x, y, z} p(x, y, z) \log \frac{p(y|x, z)}{p(y|z)}.$$

### 4.2. Свойства и тождества
- $I(X; Y | Z) = H(Y|Z) - H(Y|XZ)$.
- **Правило цепочки для взаимной информации:**
    $$I(X; YZ) = I(X; Y) + I(X; Z | Y).$$
    *Доказательство:* $I(X;YZ) = H(X) - H(X|YZ) = [H(X)-H(X|Y)] + [H(X|Y)-H(X|YZ)] = I(X;Y) + I(X;Z|Y)$.
- Аналогично: $I(X; YZ) = I(X; Z) + I(X; Y | Z)$.

### 4.3. Пример 2.9 (Многократное использование ДСК)
Рассмотрена ситуация, когда один бит информации передаётся трижды по ДСК (повторение). Показано, что отдельные принятые символы могут давать отрицательную условную информацию, но совместно они накапливают положительную информацию, демонстрируя эффект "балансировки".

## 5. Теорема о переработке информации (Data Processing Inequality)

Рассматривается система последовательной обработки (марковская цепочка): $X \rightarrow Y \rightarrow Z$, где $Z$ зависит от $X$ только через $Y$ (т.е. $X$ и $Z$ условно независимы при известном $Y$: $p(x,z|y) = p(x|y)p(z|y)$).

**Теорема 2.4:** Для такой системы справедливы неравенства:
1.  $$I(X; Y) \ge I(X; Z).$$
2.  $$I(Y; Z) \ge I(X; Z).$$

*Доказательство (для первого неравенства):*
Используем цепное правило двумя способами:
$$I(X; YZ) = I(X; Y) + I(X; Z|Y).$$
$$I(X; YZ) = I(X; Z) + I(X; Y|Z).$$
В силу условной независимости $I(X; Z|Y) = 0$. Приравнивая правые части, получаем:
$$I(X; Y) = I(X; Z) + I(X; Y|Z).$$
Поскольку $I(X; Y|Z) \ge 0$, то $I(X; Y) \ge I(X; Z)$.

**Интерпретация:**
- Никакая обработка данных (детерминированная или случайная) не может увеличить количество информации об исходном сигнале.
- В последовательности "канал $\rightarrow$ обработка" обработка на выходе не добавляет новой информации о входе, а только теряет её.
- То же верно для обработки на входе.
- Для последовательного соединения двух каналов пропускная способность всей цепи не превышает пропускной способности каждого из каналов в отдельности.

## 6. Введение в кодирование источников с памятью

В лекции ставится проблема эффективного кодирования для источников с памятью (зависимыми символами).
- Если символы зависимы, то энтропия последовательности $H(X_1 \dots X_n)$ меньше суммы энтропий отдельных символов (свойство суб-аддитивности).
- Это даёт два подхода к сжатию:
    1.  Использовать условные энтропии (кодировать каждый следующий символ, зная предыдущие) — $H(X_{n+1} | X_1 \dots X_n)$.
    2.  Кодировать блоки символов целиком, рассматривая их как буквы нового, расширенного алфавита.
- Вопрос о том, какой подход эффективнее, будет рассмотрен в следующих лекциях, посвящённых неравномерному кодированию.

# Конспект лекции №12: Кодирование для каналов с шумом. Модели каналов, пропускная способность, неравенство Фано

## 1. Постановка задачи помехоустойчивого кодирования

### 1.1. Основная дилемма
- **Скорость кода** (количество информации, передаваемое за единицу времени) и **вероятность ошибки** — два противоречивых требования.
- Увеличение числа кодовых слов (повышение скорости) приводит к:
    - уменьшению расстояния между кодовыми словами;
    - уменьшению размеров решающих областей декодера;
    - увеличению вероятности ошибки.

### 1.2. Пропускная способность канала
**Определение:** Число $C$ называется **пропускной способностью канала**, если:
1.  Для любой скорости $R < C$ существуют коды, обеспечивающие сколь угодно малую вероятность ошибки.
2.  Для любой скорости $R > C$ существует константа $\varepsilon > 0$ такая, что вероятность ошибки любого кода ограничена снизу величиной $\varepsilon$ (т.е. её нельзя сделать сколь угодно малой).

## 2. Математические модели дискретных каналов

### 2.1. Канал без памяти (ДПК)
**Определение:** Канал называется **каналом без памяти**, если для любых последовательностей $x_{j+1}^{j+n}$ и $y_{j+1}^{j+n}$ выполняется:
$$p(y_{j+1}^{j+n} \mid x_{j+1}^{j+n}) = \prod_{i=j+1}^{j+n} p(y_i \mid x_i).$$
События, происходящие при передаче последовательных символов, независимы.

**Дискретный постоянный канал (ДПК):** Стационарный канал без памяти (условные вероятности не зависят от времени).
- Полностью описывается **матрицей переходных вероятностей** $P = \{p(y|x)\}$, $|X| = K$, $|Y| = L$:
  $$P = \begin{pmatrix}
  p_{00} & p_{01} & \cdots & p_{0,L-1} \\
  p_{10} & p_{11} & \cdots & p_{1,L-1} \\
  \vdots & \vdots & \ddots & \vdots \\
  p_{K-1,0} & p_{K-1,1} & \cdots & p_{K-1,L-1}
  \end{pmatrix}.$$

### 2.2. Пример 4.3a: Двоичный симметричный канал (ДСК)
- $X = Y = \{0, 1\}$.
- Переходные вероятности: $p(1|0) = p(0|1) = p$ (вероятность ошибки), $p(0|0) = p(1|1) = 1-p$.
- Матрица переходных вероятностей:
  $$P = \begin{pmatrix} 1-p & p \\ p & 1-p \end{pmatrix}.$$

### 2.3. Пример 4.3b: Двоичный симметричный канал со стираниями (ДСтК)
- Вход: $X = \{0, 1\}$.
- Выход: $Y = \{0, 1, z\}$, где $z$ — символ стирания.
- Переходные вероятности: 
    - $p(0|0) = p(1|1) = 1 - p - \varepsilon$,
    - $p(1|0) = p(0|1) = p$ (ошибка),
    - $p(z|0) = p(z|1) = \varepsilon$ (стирание).
- Матрица:
  $$P = \begin{pmatrix} 1-p-\varepsilon & \varepsilon & p \\ p & \varepsilon & 1-p-\varepsilon \end{pmatrix}.$$

**Практическая интерпретация:**
- **ДСК:** жёсткое решение (демодулятор сравнивает отсчёт с порогом $T$).
- **ДСтК:** вводится «зона стирания» $[T-\Delta, T+\Delta]$; решения в этой зоне не принимаются. Исправлять стирания примерно в 2 раза легче, чем ошибки.
- **Канал с мягкими решениями (полунепрерывный):** выходом является непрерывная величина (например, амплитуда сигнала), что даёт больше информации для декодера.

## 3. Информационная ёмкость канала

### 3.1. Скорость передачи при блочном кодировании
- При использовании кодов длины $n$ количество информации о блоке $X^n$, содержащееся в блоке $Y^n$, равно $I(X^n; Y^n)$.
- Скорость передачи информации: $\frac{1}{n} I(X^n; Y^n)$ бит/символ канала.
- Входное распределение $p(x)$ можно выбирать для максимизации этой величины.

**Определение:** **Информационная ёмкость канала**:
$$C_0 = \sup_n \max_{p(x)} \frac{1}{n} I(X^n; Y^n).$$
(Supremum, а не max, т.к. максимум может достигаться в пределе при $n \to \infty$).

### 3.2. Теорема об информационной ёмкости ДПК (теорема 4.3)
Для дискретного постоянного канала (стационарный, без памяти):
$$C_0 = \max_{p(x)} I(X; Y).$$
Вычисление сводится к максимизации взаимной информации между **отдельными** символами по одномерным распределениям.

## 4. Неравенство Фано

### 4.1. Постановка
Рассматривается система: источник $U$ → кодер → канал → декодер → оценка $V$.
- $U$ и $V$ — множества сообщений размера $M$ (обычно $U = V = \{0, \dots, M-1\}$).
- Ошибка: $u \neq v$.
- Вероятность ошибки: $P_e = \sum_u \sum_{v \neq u} p(u,v)$.
- Вероятность правильного решения: $P_c = 1-P_e$.

**Теорема 4.1 (Неравенство Фано):**
$$H(U|V) \le \eta(P_e) + P_e \log(M-1),$$
где $\eta(P_e) = -P_e \log P_e - (1-P_e)\log(1-P_e)$ — энтропия двоичного ансамбля.

### 4.2. Обсуждение
- Левая часть $H(U|V)$ — остаточная неопределённость о переданном сообщении после получения оценки $V$.
- Правая часть $\gamma(P_e) = \eta(P_e) + P_e\log(M-1)$:
    - Строго выпукла вверх ($\cap$).
    - $\gamma(0) = 0$, $\gamma(1) = \log(M-1)$.
    - Максимум в точке $P_e = (M-1)/M$, $\gamma_{\max} = \log M$.
- **Следствие:** Для любого $\delta > 0$ существует $\varepsilon > 0$ такое, что если $\gamma(P_e) > \delta$, то $P_e > \varepsilon$.

---

## 5. Обратная теорема кодирования для каналов

**Теорема 4.2 (Обратная):** Для дискретного стационарного канала с информационной ёмкостью $C_0$ для любого $\delta > 0$ существует $\varepsilon > 0$ такое, что для любого кода со скоростью $R > C_0 + \delta$ вероятность ошибки $P_e \ge \varepsilon$.

**Смысл:** Превысить информационную ёмкость канала со сколь угодно малой ошибкой **невозможно**.

*Ключевые моменты доказательства (упомянуты в лекции):*
- Сообщения считаются равновероятными (не ограничивает общности).
- Используются свойства условной энтропии и неравенство Фано.
- В предположении $R > C_0$ удаётся показать, что $H(U|V)$ не может быть достаточно малой, откуда следует положительная нижняя граница для $P_e$.

## 6. Симметричные каналы

Для упрощения вычисления $C_0$ вводятся классы симметричных каналов.

### 6.1. Определения
- **Симметричный по входу:** Все строки матрицы переходных вероятностей получаются перестановкой элементов первой строки.
- **Симметричный по выходу:** Все столбцы матрицы получаются перестановкой элементов первого столбца.
- **Полностью симметричный:** Симметричен и по входу, и по выходу.

### 6.2. Свойства
1. Для симметричного по входу канала условная энтропия $H(Y|x)$ **одинакова** для всех $x$.
   $$C_0 = \max_{p(x)} H(Y) - H(Y|x).$$
2. Для такого канала: $C_0 \le \log M - H(Y|x)$.
3. Для симметричного по выходу канала при равновероятном входе выход равновероятен.
4. Для **полностью симметричного** канала:
   $$C_0 = \log |Y| - H(Y|x), \quad \forall x \in X.$$

### 6.3. Пример 4.5: Информационная ёмкость ДСК
Для ДСК: $|Y| = 2$, $H(Y|x) = \eta(p) = -p\log p - (1-p)\log(1-p)$.
$$C_0 = 1 - \eta(p).$$
- При $p=0$ (идеальный канал): $C_0 = 1$ бит/символ.
- При $p=0.5$ (полностью зашумлён): $C_0 = 0$.
- При $p=0.11$: $C_0 = 0.5$ бит/символ.

## Итоги лекции

1.  **Пропускная способность $C$** — фундаментальная характеристика канала, разделяющая области скоростей, где возможна надёжная передача ($R < C$) и где она невозможна ($R > C$).
2.  **Дискретный постоянный канал (ДПК)** описывается матрицей переходных вероятностей; для него информационная ёмкость $C_0 = \max_{p(x)} I(X; Y)$.
3.  **Неравенство Фано** связывает условную энтропию $H(U|V)$, вероятность ошибки $P_e$ и размер алфавита $M$; оно является ключевым инструментом для доказательства обратных теорем.
4.  **Обратная теорема кодирования** утверждает, что при $R > C_0$ вероятность ошибки не может быть сколь угодно малой.
5.  Для **симметричных каналов** (в частности, для ДСК) удаётся получить явные формулы для пропускной способности: $C_0 = 1 - \eta(p)$ для ДСК.
6.  **Каналы со стираниями** позволяют получить более высокую надёжность при той же скорости, поскольку исправлять стирания легче, чем ошибки.

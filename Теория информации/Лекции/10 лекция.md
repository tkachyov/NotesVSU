# Конспект лекции №10: Неравенство Крафта и теоремы побуквенного кодирования

## 1. Неравенство Крафта (продолжение)

### 1.1. Достаточность неравенства Крафта
Если набор длин $\ell_1, \dots, \ell_M$ удовлетворяет неравенству $\sum 2^{-\ell_i} \le 1$, то существует префиксный код с такими длинами.

**Конструктивное доказательство (идея):**
1. Упорядочиваем длины по возрастанию.
2. На ярусе $\ell_1$ выбираем любую вершину, делаем её концевой (первое кодовое слово).
3. Исключаем всё поддерево, начинающееся в этой вершине.
4. На ярусе $\ell_2$ остаётся $2^{\ell_2} - 2^{\ell_2 - \ell_1} \ge 1$ свободных вершин (следует из неравенства Крафта).
5. Повторяем процесс для всех слов.

**Следствие:** Неравенство Крафта превращается в равенство тогда и только тогда, когда кодовое дерево **полное** (каждая промежуточная вершина имеет ровно 2 потомка).

### 1.2. Неравенство Крафта для любых однозначно декодируемых кодов
**Теорема 3.1:** Для любого однозначно декодируемого двоичного кода с длинами $\ell_1, \dots, \ell_M$ справедливо:
$$\sum_{i=1}^{M} 2^{-\ell_i} \le 1.$$

**Доказательство:**
- Рассматривается $N$-я степень суммы:
  $$\left(\sum 2^{-\ell_i}\right)^N = \sum_{L=1}^{N\ell_{\max}} A_L 2^{-L},$$
  где $A_L$ — количество способов выбрать последовательность из $N$ кодовых слов с суммарной длиной $L$.
- Так как последовательность из $N$ кодовых слов — это некоторая двоичная строка длины $L$, то $A_L \le 2^L$.
- Получаем: $\left(\sum 2^{-\ell_i}\right)^N \le N\ell_{\max}$.
- Извлекая корень $N$-й степени и устремляя $N \to \infty$, получаем $\sum 2^{-\ell_i} \le 1$.

## 2. Прямая теорема побуквенного кодирования

### 2.1. Формулировка
**Теорема 3.2:** Для ансамбля $X = \{x, p(x)\}$ с энтропией $H$ существует побуквенный неравномерный префиксный код, для которого средняя длина кодового слова $\bar{\ell}$ удовлетворяет:
$$\bar{\ell} < H + 1.$$

### 2.2. Необходимые пояснения
- Доказательство **неконструктивное** (только существование).
- Выбираются длины $\ell_m = \lceil -\log p_m \rceil$.
- Неравенство Крафта выполняется:
  $$\sum 2^{-\ell_m} \le \sum 2^{\log p_m} = \sum p_m = 1.$$
- Оценка средней длины:
  $$\bar{\ell} = \sum p_m \ell_m < \sum p_m (-\log p_m + 1) = H + 1.$$

### 2.3. Важное замечание о погрешности
- Для больших $H$ (например, $H \approx 8$ бит) лишний 1 бит — незначительная потеря.
- Для малых $H$ (например, $H = 0.1$ бит) $\bar{\ell} < 1.1$ бит, но хотелось бы $\approx 0.1$ бит.
- **Проблема:** Длина кодового слова не может быть меньше 1.
- **Решение:** Переход к **кодированию блоков** символов (арифметическое кодирование позволяет приблизиться к $H$ сколь угодно близко).

## 3. Обратная теорема побуквенного кодирования

### 3.1. Формулировка
**Теорема 3.3:** Для любого однозначно декодируемого кода дискретного источника с энтропией $H$ средняя длина кодовых слов удовлетворяет:
$$\bar{\ell} \ge H.$$

### 3.2. Доказательство
$$H - \bar{\ell} = \sum p(x) \log \frac{2^{-\ell(x)}}{p(x)} \le \log e \sum p(x) \left(\frac{2^{-\ell(x)}}{p(x)} - 1\right) =$$
$$= \log e \left( \sum 2^{-\ell(x)} - 1 \right) \le \log e (1 - 1) = 0.$$

Использованы:
- Неравенство $\log t \le (t-1)\log e$,
- Неравенство Крафта $\sum 2^{-\ell(x)} \le 1$,
- Условие нормировки $\sum p(x) = 1$.

### 3.3. Условия достижения равенства
- Необходимо $p(x) = 2^{-\ell(x)}$ для всех $x$ и $\sum 2^{-\ell(x)} = 1$.
- Тогда код является **полным** и оптимальным, $\bar{\ell} = H$.

## 4. Свойства оптимальных префиксных кодов

Пусть $C$ — оптимальный префиксный код (минимальная средняя длина) для ансамбля с вероятностями $p_1 \ge p_2 \ge \dots \ge p_M$.

**Свойство 1 (монотонность):** Если $p_i < p_j$, то $\ell_i \ge \ell_j$.
- *Идея доказательства:* иначе перестановка кодовых слов уменьшила бы среднюю длину.

**Свойство 2 (два максимальных слова):** Существуют по крайней мере два кодовых слова максимальной длины $\ell_{\max}$.
- *Идея:* иначе последнее слово можно было бы укоротить.

**Свойство 3 (структура максимальных слов):** Среди слов длины $\ell_{\max}$ найдутся два, различающиеся только в последнем символе (братья в кодовом дереве).

**Свойство 4 (редукция):** Если в оптимальном коде два слова максимальной длины соответствуют наименее вероятным буквам $p_{M-1}$ и $p_M$, то после их "слияния" в один узел и перенумерации вероятностей (сложение) код для нового ансамбля также оптимален.

## 5. Алгоритм Хаффмена

Основан на свойствах 2–4:
1. Два наименее вероятных символа объединяются в один (вероятность = сумме).
2. Процесс повторяется, пока не останется один символ.
3. Кодовое дерево строится "снизу вверх": при объединении добавляется один бит (0 или 1) к каждому из двух слов.

**Пример 3.4:** $X = \{a,b,c,d,e,f\}$, вероятности $\{0.35, 0.2, 0.15, 0.1, 0.1, 0.1\}$.
- $H = 2.4016$ бит, $\bar{\ell} = 2.45$ бит.
- Избыточность $r = 0.0484$ (намного меньше 1).

### 5.1. Оценка избыточности (теорема Галлагера)
**Теорема 3.4:** Пусть $p_1$ — наибольшая вероятность. Тогда избыточность $r = \bar{\ell} - H$ удовлетворяет:
$$
r \le \begin{cases}
p_1 + \sigma, & p_1 < 1/2, \\
2 - \eta(p_1) - p_1, & p_1 \ge 1/2,
\end{cases}
$$
где $\sigma = 1 - \log e - \log \log e \approx 0.08607$, а $\eta(p_1)$ — энтропия двоичного ансамбля.

## 6. Код Шеннона (субоптимальный)

### 6.1. Идея
- Упорядочить буквы по убыванию вероятностей: $p_1 \ge p_2 \ge \dots \ge p_M$.
- Вычислить кумулятивные вероятности:
  $$q_1 = 0,\quad q_2 = p_1,\quad q_3 = p_1 + p_2,\quad \dots$$
- Длина кодового слова для $m$-й буквы: $\ell_m = \lceil -\log p_m \rceil$.
- Кодовое слово — первые $\ell_m$ битов двоичного представления числа $q_m$.

### 6.2. Свойства
- Средняя длина $\bar{\ell} < H + 1$ (как и у любого кода из прямой теоремы).
- Избыточность обычно больше, чем у кода Хаффмена.
- **Преимущество:** Простота построения (не требует итеративного "слияния").

---

## 7. Переход к блочному кодированию

- При побуквенном кодировании каждый символ заменяется целым числом бит, поэтому для малых $H$ эффективность низкая.
- При кодировании **блоков** длины $n$ энтропия блока $H(X^n) = n H$ (для источника без памяти), и теоремы кодирования дают $\bar{\ell}_{\text{block}} < nH + 1$, то есть **на один символ**:
  $$\frac{\bar{\ell}_{\text{block}}}{n} < H + \frac{1}{n}.$$
- При $n \to \infty$ можно сколь угодно близко подойти к $H$, даже если $H$ мала.

---

## Итоги лекции

1. **Неравенство Крафта** ($\sum 2^{-\ell_i} \le 1$) является необходимым и достаточным условием существования префиксного (и вообще однозначно декодируемого) кода.
2. **Прямая теорема:** всегда существует код с $\bar{\ell} < H + 1$ (неконструктивно).
3. **Обратная теорема:** для любого однозначно декодируемого кода $\bar{\ell} \ge H$.
4. **Оптимальные коды** удовлетворяют свойствам монотонности, наличия двух максимальных слов и возможности редукции.
5. **Алгоритм Хаффмена** строит оптимальный код; его избыточность на практике часто мала (теорема Галлагера).
6. **Код Шеннона** проще, но не оптимален; используется как промежуточный шаг к арифметическому кодированию.
7. **Блочное кодирование** позволяет преодолеть ограничение $\bar{\ell} \ge 1$ и приблизиться к $H$ сколь угодно точно при $n \to \infty$.

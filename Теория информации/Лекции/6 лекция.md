# Конспект лекции №6: Случайные последовательности, цепи Маркова и энтропия стационарных источников

## 1. Введение: Два подхода к экономному кодированию

Для источников с памятью (зависимыми символами) существуют две стратегии уменьшения затрат на передачу:

1.  **Учёт предыстории:** Кодировать каждый следующий символ, используя знание предыдущих. При этом затраты на $n+1$-й символ составляют $H(X_{n+1} | X_1, \dots, X_n)$ бит, что меньше или равно безусловной энтропии $H(X_{n+1})$.
2.  **Блочное кодирование:** Объединять буквы в блоки длины $n$ и кодировать эти блоки как буквы нового "расширенного" алфавита. Из суб-аддитивности энтропии $H(X_1 \dots X_n) \le \sum H(X_i)$ следует, что такой подход также экономит ресурсы.

Вопрос, какой подход эффективнее, приводит к необходимости более глубокого анализа стационарных случайных процессов.

## 2. Дискретные случайные процессы

### 2.1. Основные определения
Рассматриваются **дискретные процессы с дискретным временем** — последовательности $x_1, x_2, \dots$, где $x_i \in X$ (конечный алфавит). Процесс полностью задаётся совместными распределениями $p(x_1, \dots, x_n)$ для всех $n$.

- **Независимые одинаково распределённые (н.о.р.) процессы:**
    $$p(x_1, \dots, x_n) = \prod_{i=1}^{n} p(x_i).$$
    Такой источник называется **дискретным постоянным источником (ДПИ)**.

### 2.2. Стационарность
**Определение:** Процесс называется **стационарным**, если для любых $n$ и $t$:
$$p(x_1, \dots, x_n) = p(x_{1+t}, \dots, x_{n+t}).$$
Иными словами, вероятностные характеристики не зависят от сдвига во времени. Стационарность позволяет говорить о независимых от времени информационных характеристиках.

**Пример 2.10:** Системный генератор псевдослучайных чисел (например, `/dev/random` в Linux) может рассматриваться как источник, близкий к стационарному.

## 3. Цепи Маркова

Простейшая модель стационарного процесса с памятью.

### 3.1. Определение
Случайный процесс называется **цепью Маркова связности $s$**, если условная вероятность текущего состояния зависит только от $s$ предыдущих:
$$p(x_n | x_1, \dots, x_{n-1}) = p(x_n | x_{n-s}, \dots, x_{n-1}) \quad \text{для } n > s.$$

- **Простая цепь Маркова:** $s = 1$ (память на одно состояние).
    $$p(x_n | x_{n-1}, x_{n-2}, \dots) = p(x_n | x_{n-1}).$$

### 3.2. Описание простой цепи Маркова
Для алфавита состояний $X = \{0, 1, \dots, M-1\}$:

1.  **Начальное распределение:** $p(x_1)$.
2.  **Матрица переходных вероятностей** $\Pi$ размера $M \times M$:
    $$\pi_{ij} = P(x_t = j \mid x_{t-1} = i), \quad i, j \in X.$$
    Свойства: $\pi_{ij} \ge 0$, $\sum_{j=0}^{M-1} \pi_{ij} = 1$ (стохастическая матрица).

### 3.3. Эволюция распределений
Вектор распределения вероятностей в момент времени $t$: $p_t = (p_t(0), \dots, p_t(M-1))$.
$$p_{t+1} = p_t \Pi, \quad p_{t+n} = p_t \Pi^n.$$

### 3.4. Стационарное распределение
Распределение $p$ называется **стационарным**, если $p = p \Pi$.
Если процесс стартует со стационарного распределения, то он является стационарным (все $p_t = p$).

### 3.5. Финальное распределение и эргодичность
**Финальное распределение:**
$$p_{\infty} = \lim_{t \to \infty} p_t = \lim_{t \to \infty} p_1 \Pi^t.$$

**Теорема 2.5 (об эргодичности):** Непериодическая неприводимая цепь Маркова **эргодична**, т.е.:
- Финальное распределение $p_{\infty}$ существует.
- Оно не зависит от начального распределения $p_1$.
- Оно совпадает с единственным стационарным распределением $p$.

*Ключевые понятия:*
- **Неприводимость:** Все состояния достижимы друг из друга.
- **Непериодичность:** Отсутствие состояний с периодом $t > 1$ (т.е. возврат возможен за разное число шагов).

## 4. Энтропия стационарного источника

Для стационарного источника вводятся меры, учитывающие память.

### 4.1. Определения
- **Одномерная энтропия:** $H_1(X) = H(X)$ — энтропия одного символа.
- **$n$-мерная энтропия:** $H(X^n) = H(X_1, \dots, X_n)$ — энтропия блока длины $n$ (не зависит от положения блока в силу стационарности).
- **Энтропия на букву для блока длины $n$:**
    $$H_n(X) = \frac{H(X^n)}{n}.$$
- **Условная энтропия с предысторией:**
    $$H(X | X^{n-1}) = H(X_n | X_1, \dots, X_{n-1}).$$

### 4.2. Теорема 2.6 (Свойства энтропии стационарного источника)
Для дискретного стационарного источника справедливо:

1.  **A.** $H(X | X^n)$ не возрастает с ростом $n$ (больше предыстории — меньше неопределённость).
2.  **B.** $H_n(X)$ не возрастает с ростом $n$ (чем длиннее блок, тем меньше средняя энтропия на символ).
3.  **C.** $H_n(X) \ge H(X | X^{n-1})$ (условная энтропия даёт более оптимистичную оценку, чем блочная).
4.  **D.** Существуют пределы:
    $$\lim_{n \to \infty} H_n(X) = \lim_{n \to \infty} H(X | X^{n-1}) \triangleq H_{\infty}.$$

*Доказательство основано на свойствах условной энтропии, стационарности и неравенстве Йенсена.*

### 4.3. Интерпретация
- Величина $H_{\infty}$ называется **энтропией на символ** (entropy rate) или **скоростью порождения информации** источником.
- Это фундаментальная нижняя граница среднего количества бит, необходимого для представления одного символа источника при кодировании без потерь.
- Два подхода к кодированию (блочное кодирование и кодирование с учётом предыстории) в пределе дают одинаковый результат: $H_{\infty}$.

## 5. Примеры вычисления $H_{\infty}$

### 5.1. Пример 2.11: Дискретный постоянный источник (без памяти)
Для источника без памяти:
- $H(X^n) = n H(X)$.
- $H_n(X) = H(X)$ для всех $n$.
- $H(X | X^{n-1}) = H(X)$.
- Следовательно, $H_{\infty} = H(X)$.

*Важное замечание:* Даже для независимых символов кодирование блоков может быть эффективнее побуквенного (например, арифметическое кодирование).

### 5.2. Пример 2.12: Марковский источник (связности $s$)
Для стационарной цепи Маркова связности $s$:
- Условная энтропия стабилизируется при $n \ge s$:
    $$H(X | X^{n-1}) = H(X | X^s) \quad \text{для } n > s.$$
- $n$-мерная энтропия:
    $$H(X^n) = s H_s(X) + (n-s) H(X | X^s).$$
- В пределе:
    $$H_{\infty} = H(X | X^s).$$

*Формула для $H_n(X)$:*
$$H_n(X) = H(X|X^s) + \frac{s}{n}\bigl(H_s(X) - H(X|X^s)\bigr).$$
"Потери" $\frac{s}{n}\bigl(H_s(X) - H(X|X^s)\bigr)$ связаны с тем, что первые $s$ символов блока кодируются без полной предыстории (краевой эффект). При $n \to \infty$ эти потери исчезают.

---

## 6. Итоги и выводы

1.  **$H_{\infty}$** — ключевая характеристика стационарного источника, определяющая минимально возможные удельные затраты на передачу (сжатие).
2.  Для источника **без памяти**: $H_{\infty} = H(X)$.
3.  Для **марковского источника** связности $s$: $H_{\infty} = H(X | X^s)$.
4.  Блочное кодирование ($H_n(X)$) и кодирование с учётом предыстории ($H(X|X^{n-1})$) дают разные оценки при конечных $n$, но сходятся к одному пределу $H_{\infty}$ при $n \to \infty$.
5.  Теорема об эргодичности гарантирует, что для широкого класса процессов (включая непериодические неприводимые цепи Маркова) временны́е средние сходятся к ансамблевым, что важно для практического кодирования.

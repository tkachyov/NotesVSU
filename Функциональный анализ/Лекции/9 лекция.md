#### Линейный функционал

$L$ - произвольное линейное пространство.

**Определение.**
Отображение $f$, сопоставляющее всякому элементу $L$ число, называется функционалом.


Функционал называется линейным, если выполняется:
1) $f(x + y) = f(x) + f(y)$ - аддитивность
2) $f(\alpha x) = \alpha f(x)$ - однородность


Примеры линейных пространств:
1) $C_{[a;b]}$
$$f (x(t)) = \int_a^b x(t) dt$$
2) $\mathbb{R}^n, x \in \mathbb{R}^n, \bar{a} \in \mathbb{R}^n$
$$f (x) = (\bar{a}, \bar{x})$$


##### Геометрический смысл линейного функционала в евклидовом пространстве.

Рассмотрим $\mathbb{R}^n$ и произвольный функционал $f(x)$ - ненулевой функционал.

**Определение.**
$f(x)$ - нулевой функционал, если ставит в соответствии каждому элементу число 0.

$$\exists y: f(y) \neq 0$$
$$Ker f \equiv \{x: f(x) \}$$


**Определение.**
Ядром функционала называется множество таких элементов, что значение функционала на элементах этого множества равно 0.


**Утверждение.**
Ядро линейного функционала есть подпространство пространства $\mathbb{R}^n$ размерности на единицу меньше ($dim(Ker f) = n-1$).


Пусть $f(y) = 1$, если это не так, то:
$$f(\frac{y}{f(y)}) = 1 (= \frac{f(y)}{f(y)} = 1)$$

$y_0 = \frac{y}{f(y)}$ на этом элементе $f(y_0) = 1$

Зафиксируем произвольный элемент $x$.

$$x \rightarrow x - f(x) y_0 = x_0$$
$$f(x_0) = f(x - f(x)y_0) = - f(x) - f(x) f(y_0) = 0$$

Следовательно, $x_0 \in ker f$, 
$$x = f(x) y_0 + x_0 \tag{*}$$


В пространстве $\mathbb{R}^n$ выберем базис, содержащий элемент $y_0$.

$$x = \sum_{i=0}^{n-1} \alpha_i y_i = \alpha_0 y_0 + \sum_{i=1}^{n-1} \alpha_i y_i = $$
В силу теоремы о единственности разложения элемента по базису: $\alpha_0 = f(x)$.

$$= f(x) y_0 + \sum_{i=1}^{n-1} \alpha_i y_i \tag{**}$$


$$x_0 = \sum_{i=1}^{n-1} \alpha_i y_i \implies dim (Ker f) = n-1$$
$$x_0 \rightarrow ker f$$

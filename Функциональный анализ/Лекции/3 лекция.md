Рассмотрим Евклидову метрику:
$$\mathbb{R}^n, \rho (x, y) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}$$
$$x_n \Rightarrow x_0$$
$$\forall \varepsilon > 0,  \exists N, \forall n > N: \rho(x_n, x_0) < \varepsilon$$

$$\sqrt{\sum_{i=1}^m (x_i^n - x_i^0)^2} < \varepsilon \Rightarrow \sum_{i=1}^m (x_i^n - x_i^0)^2 < \varepsilon^2 \Rightarrow \forall i = 1, 2, ..., m: (x_i^n - x_i^0)^2 < \varepsilon^2$$
$$(x_i^n - x_i^0)^2 < \varepsilon \tag{1}$$
$$\{x^n_i\} \underset{n \to \infty}{\longrightarrow} x_i^0$$

Из $(1)$ следует, что численная последовательность $\{x^n_i\}$ сходится к $x_i^0$
Таким образом, сходимость в Евклидовом пространстве $\mathbb{R}^n$ означает покоординатную сходимость.

!!!Доказательство аналогичного результата в пространстве $L^2$ с аналогичной метрикой.

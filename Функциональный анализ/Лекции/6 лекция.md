#### Линейные пространства

Множество $L$ называется линейным пространством, если для любых двух элементов $\forall x, y \in L$: $X + Y \in L$.
$\forall x: \forall \lambda$: $\lambda x \in L$.

**Свойства:**
1) $x + y = y + x$
2) (x + y) + z = x + (y + z)
3) $\exists o + x = x$
4) $\forall x:$ $\exists (-x)$: $x + (-x) = 0$

Операция со скаляром удовлетворяет следующим свойствам:
1) $(\alpha + \beta)x = \alpha x + \beta x$
2) $\alpha (\beta x) = \alpha \beta x$
3) $1 \cdot x = x$


**Примеры линейных пространств**

1) $\mathbb{R}^n$ 
$$x = \begin{pmatrix}
x_1 \\
x_2 \\
... \\
x_n
\end{pmatrix}$$

2) $C_{[a;b]}$
3) $l_p, l_{\infty}, L_p, L_{\infty}$


Есть пространства $L$ и $\tilde{L}$.
Отображение $\varphi: L \to \tilde{L}$ называется изоморфизмом, если 
$$\varphi (x + y) = \varphi (x) + \varphi (y)$$
$$\varphi (\alpha x) = \alpha \varphi(x)$$

**Пример:**
Есть $\mathbb{R}^n$ и $A_{n-1}$ - пространство многочленов в степени $(n-1)$.



Пусть $x_1, x_2, ..., x_n \in L$.
Составим линейную комбинацию:
$$\lambda_1 x_1 + \lambda_2 x_2 + ... + \lambda_n x_n = o$$ o - нулевой элемент пространства $L$.

Если это равенство выполняется, то вектор называется линейно-независимым.

Размерностью линейного пространства $L$ называется такое натуральное число $N$, что существует набор из $x_1, x_2, ..., x_n$ - ЛНЗ, а $x_1, x_2, ..., x_{n+1}$ - ЛЗ.

Базисом называется произвольный набор из максимального числа ЛНЗ векторов.


Есть $\lambda_1 x_1 + \lambda_2 x_2 + ... + \lambda_n x_n = o$.
$dim L = n$ - в пространстве $L$ найдётся $n$ ЛНЗ векторов.


Пусть задано линейное пространство $L$ ($\tilde{L} \subset L$).
$\tilde{L}$ является линейным многообразием пространства $L$, если оно замкнуто относительно операции $+$ и $\cdot$ на скаляры.

**Примеры**
1) $A_n$ - множество многочленов степени $n$.
	$A_n \subset C_{[a, b]}$
2) $\mathbb{R}^3$: 
$$\tilde{L} = \{x: x = (o, x_2, x_3) \}$$


##### Линейные нормированные пространства

Пространство $L$ называется линейным нормированным пространством, если каждому элементу $L$ сопоставляется $x \to ||x||$
1) $$||x|| > 0$$ $$||x|| = 0 \leftrightarrow x = o$$
2) $$||\alpha x|| = |\alpha| \cdot ||x||$$
3) $$||X + Y|| \le ||X|| + ||Y||$$

Всякое линейное нормированное пространство является метрическим, если ввесит метрику следующим образом: 
$$\rho (x - y) = ||X - Y||$$

Пример:
$$\rho (x, y) = 
\begin{cases}
    1, &\text{если } x \neq y \\
    0,   &\text{если } x = y
\end{cases}$$

Не всякой метрике соответствует норма.



##### Евклидово пространство. Скалярное произведение.

Билинейная форма называется скалярным произведением, если:
1) $$(x, y) = (y, x)$$
2) $$(x, y + z) = (xy) + (xz)$$
3) $$(\alpha x, y) = \alpha (x, y)$$
4) $$(x, x) \ge 0, (x, x) = 0 \leftrightarrow x = 0$$


**Примеры линейных нормированных пространств**

1) $C_{[a, b]}$, $||x(t)|| = \underset{t}{max} |x(t)|$
2) $l_p$: $||x|| = \sum_{i = 1}^{\infty} |x_i|^p$
	$l_{\infty}$: $||x|| = \underset{i}{sup} |x_i|$
3) $L_p$: $L_p[a,b] = \left\{ f:\ [a,b] \to \mathbb{R} \ \Big|\ \int_a^b |f(x)|^p dx < \infty \right\}, \quad 1 \le p < \infty$
	$L_{\infty}$: $L_\infty[a,b] = \left\{ f:\ [a,b] \to \mathbb{R} \ \Big|\ \sup_{x \in [a,b]} |f(x)| < \infty \right\}.$
4) $\mathbb{R}^n$ 
	1) $||x|| = \sqrt{\sum x_i^2}$
	2) $||x|| = max|x_i|$
	3) $||x|| = \sum |x_i|$

$||x|| = \sqrt{\sum x_i^2}$
$\textbf{Доказательство.}$

1) Положительность:

$$
||x|| = \sqrt{\sum x_i^2} \ge 0
$$

и

$$
||x|| = 0 \Leftrightarrow x_i = 0 \ \forall i \Rightarrow x = 0.
$$

2) Однородность:

$$
||\alpha x|| = \sqrt{\sum (\alpha x_i)^2}
= \sqrt{\alpha^2 \sum x_i^2}
= |\alpha| \cdot ||x||.
$$

3) Неравенство треугольника:

$$
||x+y||^2 = \sum (x_i + y_i)^2
$$

$$
= \sum x_i^2 + 2\sum x_i y_i + \sum y_i^2
$$

По неравенству Коши–Буняковского:

$$
\sum x_i y_i \le \sqrt{\sum x_i^2} \cdot \sqrt{\sum y_i^2}
$$

Следовательно,

$$
||x+y||^2 \le ||x||^2 + 2||x|| \cdot ||y|| + ||y||^2
= (||x|| + ||y||)^2
$$

$$
\Rightarrow ||x+y|| \le ||x|| + ||y||.
$$

Следовательно, это норма.



$||x|| = max|x_i|$
$\textbf{Доказательство.}$

1) Положительность:

$$
||x|| = \max |x_i| \ge 0
$$

и

$$
||x|| = 0 \Leftrightarrow x_i = 0 \Rightarrow x = 0.
$$

2) Однородность:

$$
||\alpha x|| = \max |\alpha x_i| = |\alpha| \max |x_i| = |\alpha| ||x||.
$$

3) Неравенство треугольника:

$$
|x_i + y_i| \le |x_i| + |y_i|
$$

Следовательно

$$
\max |x_i + y_i| \le \max (|x_i| + |y_i|)
\le \max |x_i| + \max |y_i|
$$

$$
= ||x|| + ||y||.
$$

Следовательно, это норма.


$||x|| = \sum |x_i|$
$\textbf{Доказательство.}$

1) Положительность:

$$
||x|| = \sum |x_i| \ge 0
$$

и

$$
||x|| = 0 \Leftrightarrow x_i = 0 \ \forall i \Rightarrow x=0.
$$

2) Однородность:

$$
||\alpha x|| = \sum |\alpha x_i|
= |\alpha| \sum |x_i|
= |\alpha| ||x||.

3) Неравенство треугольника:

$$
||x+y|| = \sum |x_i + y_i|
\le \sum (|x_i| + |y_i|)
$$

$$
= \sum |x_i| + \sum |y_i|
= ||x|| + ||y||.
$$

Следовательно, это норма.

$$
||\alpha x|| = \sqrt{(\alpha x, \alpha x)}

$$
||x+y||^2 = (x+y, x+y)

$$

$$
(x,y) \le ||x|| \cdot ||y||

$$

$$

$$
= (||x|| + ||y||)^2

$$
||x+y|| \le ||x|| + ||y||.

Нахождение угла между между векторами:
$$cos (x, y) = \frac{(x, y)}{||x|| \cdot ||y||}$$

Если скалярное произведение $(a, b) = 0, (a, b \in X)$, то такие векторы называются ортогональными.



Пусть дан набор векторов $\varphi_1, \varphi_2,..., \varphi_n$, $(\varphi_i, \varphi_j) = 0$, $i \ne j$

Сегодня говорим о векторном пространстве.

Если $||\varphi_i|| = 1$, то будем называть ортонормированными.

$$\frac{\varphi_1}{||\varphi_1||} \cdot \frac{\varphi_2}{||\varphi_2||} \cdot ... \cdot \frac{\varphi_n}{||\varphi_n||}$$

Если вектор продлить на свою длину в евклидовом пространстве, то получим 1.


$E$ - Евклидово пространство (линейное нормированное полное пространство со скалярным произведением)


Пусть $\varphi_1, \varphi_2,..., \varphi_n$ - набор ортогональных векторов из $E$.


**Утверждение.** Эти векторы ЛНЗ.

**Доказательство.**

$$\lambda_1 \varphi_1 + \lambda_2 \varphi_2 + ... + \lambda_n \varphi_n = \vec{0}$$
Последовательно умножим на $\varphi_1, \varphi_2,..., \varphi_n$.

$$\lambda_1 (\varphi_1, \varphi_1) = 0$$
$$\lambda_1 = 0$$

$$\lambda_2 (\varphi_2, \varphi_2) = 0$$
$$\lambda_2 = 0$$
$$...$$
$$\lambda_1 (\varphi_n, \varphi_n) = 0$$
$$\lambda_n = 0$$
$$\lambda_1 = \lambda_2 = ... = \lambda_n$$


**Определение.**
Множество векторов $\varphi_1, \varphi_2,..., \varphi_n$ может быть до $\infty$. Эта система называется полной, если минимальное линейное пространство, вмещающее (включающее) в себя весь этот набор совпадает со всем пространством.

Пусть задано пространство $E$, $\varphi_1, \varphi_2,..., \varphi_n$ образует ортонормированный базис
$(||\varphi_1|| = ... = ||\varphi_n|| = 1)$.

$$x = \sum_{i = 1}^{\infty} x_i \varphi_i$$

Пусть $E$ - сепарабельное гильбертово (линейное нормированное пространство со скалярным произведением) пространство.

$$||x|| = \sqrt{(x, x)}$$


**Утверждение.**
Любой ортонормированный базис в сепарабельном гильбертовом пространстве, имеет не более чем счётное число элементов.


**Доказательство.**

$$||\varphi_1 - \varphi_2|| = \sqrt{(\varphi_1 - \varphi_2)(\varphi_1 - \varphi_2)} = \sqrt{(\varphi_1, \varphi_1) - (\varphi_1, \varphi_2) - (\varphi_2, \varphi_1) + (\varphi_2, \varphi_2)} = \sqrt{1 + 0 + 0 + 1}$$
$$||\varphi_1 - \varphi_2|| = \sqrt{2}$$


Выберем систему шаров с радиусом $r = \frac{1}{2}$, с центрами в базисных векторах.

По построению эти шары не пересекаются. В силу сепарабельности Евклидова пространства существует всюду счётное плотное множество, поэтому в каждом из существующих шаров найдётся элемент этого множества.

Пусть $\varphi_1, \varphi_2,..., \varphi_n$ образует базис в гильбертовом пространстве. 


**Утверждение.**
Существует ортонормированный базис $f_1, f_2, ..., f_n$ такой, что $f_n = \sum_{i=1}^n \lambda_i \varphi_i$.


**Доказательство.**

$$f_1 = \frac{\varphi_1}{||\varphi_1||}, f_2 = \alpha f_1 + \varphi_2$$
Выберем $\alpha$ таким, что $(f_2, f_1) = 0$.

$$\alpha (f_1, f_2) + (\varphi_2, f_1) = 0$$
$$\alpha = - (\varphi_2, f_1)$$

$$f_2 = \frac{\tilde{f_2}}{||f_2||}$$
$$\tilde{f_3} = \beta_1 f_1 + \beta_2 f_2 + \varphi_3$$
довести до ума ✅

$$\begin{cases}
(\tilde{f_3}, f_1) = 0 \\
(\tilde{f_3}, f_2) = 0
\end{cases}$$

$\beta_1, \beta_2$ - ?



Найдём коэффициенты.

$$
\tilde{f_3} = \beta_1 f_1 + \beta_2 f_2 + \varphi_3
$$

Условия:

$$
(\tilde{f_3}, f_1) = 0, \quad (\tilde{f_3}, f_2) = 0
$$

Тогда:

$$
\beta_1 = -(\varphi_3, f_1), \quad \beta_2 = -(\varphi_3, f_2)
$$

Следовательно,

$$
\tilde{f_3} = \varphi_3 - (\varphi_3, f_1) f_1 - (\varphi_3, f_2) f_2
$$

Нормируем:

$$
f_3 = \frac{\tilde{f_3}}{||\tilde{f_3}||}
$$

Пусть всякая последовательность вложенных шаров имеет пересечение.
$\{ x_n \}$ - фундаментальная последовательность пространства $X$.

$$\forall \varepsilon > 0,  \exists N, \forall k, m > N: \rho(x_k, x_m) < \varepsilon$$

Пусть $\varepsilon = \frac{1}{2^n}$

Есть метрика $\rho (x_k, x_m) < \frac{1}{2}$

Возьмём $\varepsilon_1$
$$\forall m \ge N_1: \rho(x_{N_1}, x_m) < \frac{1}{2}$$

$x_{N_1}$ объявим центром шара $\bar{B}(x_{N_1}, 1).$



![[7.jpg]]


Выявим $\varepsilon_2 = \frac{1}{4}, N_2 > N_1.$
$$\forall k, m \ge N_2: \rho (x_k, x_m) < \frac{1}{4}, x_k = N_2$$

Рассмотрим шар $B (x_{N_2}, \frac{1}{2})$.

Выберем $\varepsilon_3 = \frac{1}{8}$.
Возьмём шар $B (x_{N_3}, \frac{1}{4})$.


В результате получим последовательность вложенных шаров с радиусами $\longrightarrow 0$.


В силу исходных предположений эта последовательность имеет непустое пересечение, то есть оно содержит, по крайней мере, одну точку $x_0$.
Так как эта точка принадлежит всем шарам и $\rho(x_{N_p}, x_0) < r_p = \frac{1}{2^p} \underset{p \to \infty}{\longrightarrow} 0$.

Следовательно, $x_0$ является пределом последовательности, образованной центрами этих шаров.

Тем самым доказано, что $\{x_{N_p}\}$ содержит сходящиеся к элементу пространства подпоследовательности.


Докажем, что $x_N$ так же сходится.
По построению последовательности вложенных шаров все последовательности $\{x_n\}$, начиная с номера $n \ge N_p$, принадлежат шару $B(x_{N_p}, r_p)$.
Тогда 
$$\rho (x_n, x_0) \le \rho(x_n, x_{N_p}) + \rho(x_{N_p}, x_0) \le \frac{1}{2^p} + \frac{1}{2^p} = \frac{1}{2^{p-1}} \underset{p \to \infty}{\longrightarrow} 0$$
Теорема доказана.





Пусть дано произвольное множество $M \subset X$.
$\bar{M}$ - замыкание множества $M$ - все предельные точки множества $M$.

Пусть даны $Y$ и $Z$.
Множество $Y$ называется всюду плотным в множестве $Z$, если множество $Z$ включено в замыкание множества $Y$:
$$\bar{Y} \supset Z$$

$$\forall x_z \in Z: \forall \varepsilon > 0: \exists x_y \in Y: \rho(x_z, y_y) < \varepsilon$$

**Пример:**
$$M \equiv (0, 1)$$
Тогда $\bar{M} = [0, 1]$


**Пример:**
$$Q \in [0, 1]$$
тогда $\bar{Q} = R$


**Пример:**
Множество многочленов с рациональными коэффициентами плотно в множестве непрерывных заданных на конечном отрезке функций.


**Определение.**
Метрическое пространство называется сепарабельным, если оно содержит счётное 
всюду плотное в нём подмножество.


Примеры: $\mathbb{R}^n$, множество точек векторов с рациональными координатами.



### Пополнение неполных пространств.

 
Пусть задано:
$$F: X \longrightarrow Y$$

$F$ называется изометрией(изоморфизмом), если $\rho (F(x_1), F(x_2)) = \rho(x_1, x_2)$, то есть если оно сохраняет расстояния.

Пространство $Y$ называет пополнением метрического пространства $X$, если:
1) $X \subset Y$
2) $Y$ - полно


##### Теорема о пополнении метрического пространства.

Всякое метрическое пространство имеет пополнение единственное с точностью до изометрии (пополнений может быть несколько, но между ними можно установить одно).



##### Принцип сжимающихся отображений.

Рассмотрим $F: X  \longrightarrow Y$

Отображение называется непрерывным отображением в точке $z_0$, если $$\forall \varepsilon > 0: \exists \delta: \rho_Y (F_{z_0}, F(z)) < \varepsilon, \rho_X (z_0, z) < \delta$$


Рассмотрим $F: X  \longrightarrow X$

Отображение называется сжимающим, если 
$$\exists \alpha < 1: \rho(F(x_1), F(x_2)) \le \alpha \rho (x_1, x_2)$$


Рассмотрим 
$$Fx = x \tag{1}$$

Решение уравнения $(1)$ называются неподвижными точками отображения $F$.


##### Теорема (принцип сжимающих отображений)

$X$ - полное метрическое пространство.
Всякое сжимающее отображение имеет единственную неподвижную точку.


**Доказательство:**

Точка $x_0$

$$x_n = F x_{n-1}$$
$$x_1 = F x_0$$
$$x_2 = Fx_1 = F^2 x_0$$
$$...$$
$$x_n = F^{(n)} x_0 = F (F (...F(x_0))...)$$

Пусть $p > 1$, $n$ - натуральное число.

$$\rho(x_{n+p}, x_n) < \alpha \rho (F x_{n+p-1}, Fx_{n-1}) < \alpha^2 \rho (F^2 x_{n+p-2}, F_{n-2}) ...$$$$... < \alpha^{n} \rho (F^{(n)} x_p, F^{(n)}x_0) \le$$
$$\le \alpha^{n} (\rho (F^{(n)}x_p, F^{(n)}x_{p-1}) + \rho(F^{(n)}x_{p-1}, F^{(n)}x_{p-2}) + ... + \rho(F^{(n)}x_1, F^{(n)}x_0))$$


$$\rho (F^{(n)}x_1, F^{(n)}x_0) = \rho (F(F^{(n-1)}x_1), F(F^{(n-1)}x_0)) \le 
$$$$\le \alpha \rho (F^{(n-1)} x_1, F^{(n-1)}x_0) \le \alpha^2 \rho (F^{(n-2)}x_1, F^{(n-2)}x_0) ... \le \alpha^n \rho (x_1, x_0)$$

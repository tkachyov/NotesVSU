Введём метрику.
$$X \equiv \mathbb{R}^n$$
$$\rho(\vec{a}, \vec{b}) = \left(\sum_{i=1}^n |a_i - b_i|^p \right)^{\frac{1}{p}}$$  


Докажем неравенство треугольника этой метрики:
$$\sum_{i=1}^n \left(|x_i| + |y_i| \right)^p \equiv \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} \left(|x_i| + |y_i| \right)^p = \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} |x_i| + \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} |y_i| \le$$
$$\le \left(\sum_{i=1}^n \left(\left(|x_i| + |y_i| \right)^{p-1} \right)^q \right)^{\frac{1}{q}} \left(\sum_{i=1}^n |x_i|^p\right)^{\frac{1}{p}} + \left(\sum_{i=1}^n \left(\left(|x_i| + |y_i| \right)^{p-1} \right)^q \right)^{\frac{1}{q}} \left(\sum_{i=1}^n |y_i|^p \right)^{\frac{1}{p}}$$

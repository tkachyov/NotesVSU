Введём метрику.
$$X \equiv \mathbb{R}^n$$
$$\rho(\vec{a}, \vec{b}) = \left(\sum_{i=1}^n |a_i - b_i|^p \right)^{\frac{1}{p}}$$  


Докажем неравенство треугольника этой метрики:
$$\sum_{i=1}^n \left(|x_i| + |y_i| \right)^p \equiv \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} \left(|x_i| + |y_i| \right)^p = \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} |x_i| + \sum_{i=1}^n \left(|x_i| + |y_i| \right)^{p-1} |y_i| \le$$
$$\le \left(\sum_{i=1}^n \left(\left(|x_i| + |y_i| \right)^{p-1} \right)^q \right)^{\frac{1}{q}} \left(\sum_{i=1}^n |x_i|^p\right)^{\frac{1}{p}} + \left(\sum_{i=1}^n \left(\left(|x_i| + |y_i| \right)^{p-1} \right)^q \right)^{\frac{1}{q}} \left(\sum_{i=1}^n |y_i|^p \right)^{\frac{1}{p}} = $$


$$\frac{1}{p} + \frac{1}{q} = 1$$
$$1 = \frac{p-1}{p}q$$
$$(p-1)q=p$$
$$= \left( \sum_{i=1}^n (|x_i| + |y_i|)^p \right)^{\frac{1}{q}} \left( \left( \sum_{i=1}^n |x_i|^p \right)^{\frac{1}{p}} + \left( \sum_{i=1}^n |y_i|^p \right)^{\frac{1}{p}} \right)$$
$$\left( \sum (x_i + y_i)^p \right)^{\frac{1}{p}} \le \left( \sum |x_i|^p \right)^{\frac{1}{p}} + \left( \sum |y_i|^p \right)^{\frac{1}{p}}$$
Это неравенство Минковского.


Выполним замену.

$$y_i = c_i - b_i$$

По итогу пришли к неравенству треугольника:
$$\rho(a, b) \le \rho(a,c) + \rho(c, b)$$

Пусть есть метрическое пространство $X$.
Пусть для $n \to x_n$ это будет последовательность $\{x_n \}$.
Последовательность $\{x_n \}$ называется сходящейся к $x^*$, если числовая последовательность $\rho (x_n, x^*) \underset{n \to \infty}{\longrightarrow} 0 \Rightarrow x_n \underset{n \to \infty}{\longrightarrow} x^*$

$x_0$ - предельная точка последовательности $\{x_n \}$, если в любой момент её окрестности (шар радиусом $\varepsilon, R(\varepsilon, x_0)$) содержится бесконечное число членов последовательности.


Пусть есть множество $A \subset X$, $a$ - предельная точка множества $A$, если в любой её окрестности содержатся точки множества $A$.

Множество $A$ метрического пространства $X$ называется замкнутым, если оно содержит все свои предельные точки.

Множество называется открытым, если оно содержит вместе с каждой своей точкой некоторую её окрестность.

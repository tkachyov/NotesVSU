##### Теорема (О существовании и единственности решения уравнения)

$$y^{'} = f(y, t) \tag{2}$$
$$y(t_0) = y_0$$
$$|f(y_1, t) - f(y_2, t)| \le L|y_1 - y_2|, L < 1$$

Уравнение $(2)$ имеет единственное решение.



*Доказательство:*

$$\int_{t_0}^{t} y^{'} d \tau = \int_{t_0}^{t} f(y(\tau), \tau) d \tau$$
$$y(t) - y(t_0) = \int_{t_0}^{t} f(y(\tau), \tau) d \tau$$
$$y(t) = \int_{t_0}^{t} f(y(\tau), \tau) d \tau + y_0 \tag{3}$$

$(3)$ - интегральное уравнение.
$(2)$ и $(3)$ одно и то же.



Рассмотрим отображение $A : C_{[t_0 - h, t_0 + h]} \longrightarrow C_{[t_0 - h, t_0 + h]}^1.$

$C^1$ - пространство непрерывно дифференцируемых функций.


$$A: y \longrightarrow Ay$$
$$A y(t) = \int_{t_0}^{t} f(y(\tau), \tau) d \tau + y_0 \tag{4}$$


Если функция $f$ удовлетворяет условию Липшица, то отображение $A$ сопоставляет всякой непрерывной функции непрерывную функцию. Следовательно, отображение $A$ переводит себя пространство непрерывных функций. Докажем, что отображение $A$ является сжимающим при соответствующим выборе параметра $h$.

Наделим пространство непрерывных функций метрикой 
$$\rho (Ay_1, Ay_2) = \underset{t_0 - h \le t \le t_0 + h}{max}|A y_1(t) - A y_2(t)| = \underset{t_0 - h \le t \le t_0 + h}{max} \left|\int_{t_0}^{t} \left(f(y_1(\tau), \tau) - f(y_2(\tau), \tau) \right) d \tau \right| = $$
$$ = \underset{t_0 - h \le t \le t_0 + h}{max} \int_{t_0}^{t} \left| f(y_1(\tau), \tau) - f(y_2(\tau), \tau) \right| d \tau \le$$
$$\le \underset{t_0 - h \le t \le t_0 + h}{max} \int_{t_0}^{t} L |y_1(\tau) - y_2(\tau)| d \tau \le $$
$$\le L \int_{t_0}^{t} \underset{\tau}{max} |y_1(\tau) - y_2(\tau)| d \tau =$$$$= L \rho (y_1(t), y_2(t)) |t-t_0| = L \rho (y_1(t), y_2(t)) h$$

$$\rho (Ay_1, Ay_2) \le L h \rho (y_1, y_2), Lh = \alpha < 1$$

Выберем $h$ таким образом, чтобы выполнялось: $Lh < 1$.

Тогда при таком выборе $h$, отображение $(4)$ является сжимающим, и в силу принципа сжимающего отображения, имеем единственную неподвижную точку, эта неподвижная точка является решением уравнения $(3)$, которое в свою очередь эквивалентно исходному уравнению $(2)$.

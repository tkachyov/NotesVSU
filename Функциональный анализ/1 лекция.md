# Метрические Пространства. Геометрия метрических пространств.

Пусть есть множество $X$, на нём задана функция $\rho_X(x, y)$, удовлетворяющая следующим аксиомам:
1) $\rho(x, y) \ge  0: \rho(x, y) = 0 \leftrightarrow x=y$
2) $\rho(x, y) = \rho(y, x)$
3) $\rho(x, y) \le \rho(x, z) + \rho(z, y)$

Если функция удовлетворяет этим аксиомам, то она называется метрикой.

Множество $X_{\rho}$ с введённым в него функций $\rho$  называется метрическим.

В определении метрического пространства не требуется линейная структура на множестве $X$ (то есть операции сложения на множестве $X$, умножения на скаляры могут отсутствовать).

### Примеры: 
$$
\rho(x, y) =
\begin{cases}
0, & x = y \\
1, & x \ne y
\end{cases}
$$
Эта функция удовлетворяет всем аксиомам.



Есть $\mathbb{R}^n$, введём метрику 
$$
\rho_2 (x, y) = \sqrt {\sum_{i=1}^n (x_i - y_i)^2}
$$
$$
\rho_1 (x, y) = {\sum_{i=1}^n |x_i - y_i|}
$$
$$
\rho_{\infty} (x, y) = max|x_i - y_i|
$$
$$
\rho_p(x, y) = \left(\sum_{i=1}^{n} |x_i - y_i|^p\right)^{\frac{1}{p}}, (p \ge 1)
$$


Пусть $n=2$
![[1.png]]


Покажем, что $\lim_{p \to 0} \rho_p=\rho_{\infty}:$
$max|x_{i} - y_{i}| = |x_{i*} - y_{i*}|$

$$
\rho_p = \left(\sum_{i=1}^{n} |x_i - y_i|^p\right)^{\frac{1}{p}} = |x_{i*} - y_{i*}|^p \left(1 + \sum_{i \ne i*}^{n} \left|\frac{x_i - y_i}{x_{i*} - y_{i*}}\right|^p \right)^{\frac{1}{p}} \underset{p \to \infty}{\longrightarrow} |x_{i*} - y_{i*}|
$$


Необходимо доказать, что: 
$$\rho_2 (x, y) \le \rho_2 (x, z) + \rho_2 (z, y)$$

Воспользуемся неравенством Коши-Буняковского:
$$\sum a_i b_i \le \sqrt{\sum_i a_i^2 \sum_i b_i^2}$$

Таким образом, получим:
$$\sqrt{\sum_{i=1}^n (x_i - y_i)^2} \le \sqrt{\sum_{i=1}^n (x_i - z_i)^2} + \sqrt{\sum_{i=1}^n (z_i - y_i)^2}$$

Сделаем замену:
$$x_i - z_i = a_i$$
$$z_i - y_i = b_i$$
Получим:

$$\sqrt{\sum_{i=1}^n (a_i + b_i)^2} \le \sqrt{\sum_{i=1}^n a_i^2} + \sqrt{\sum_{i=1}^n b_i^2}$$

$$\sum_{i=1}^n (a_i + b_i)^2 \le \sum_{i=1}^n a_i^2 + 2\sqrt{\sum_{i=1}^n a_i^2}\sqrt{\sum_{i=1}^n b_i^2} + \sum_{i=1}^n b_i^2$$

$$\cancel{2}\sum_{i=1}^n a_i b_i \le \cancel{2} \sqrt{\sum_{i=1}^n a_i^2 \sum_{i=1}^n b_i^2}$$

Пришли к неравенству Коши-Буняковского, следовательно, доказали.



Далее, докажем:
$$\left(\sum_{i=1} |x_i - y_i|^p\right)^{\frac{1}{p}} \le \left(\sum_{i=1} |x_i - z_i|^p\right)^{\frac{1}{p}} + \left(\sum_{i=1} |z_i - y_i|^p\right)^{\frac{1}{p}}$$

Знаем, что $\frac{1}{p} + \frac{1}{q} = 1$


Докажем: $ab \le \frac{a^p}{p} + \frac{b^q}{q}$

Рассмотрим следующее:

$$\frac{1}{p} = \frac{p-1}{p} \rightarrow \frac{1}{p-1} = \frac{q}{p}$$
$$\frac{1}{p} = \frac{q-1}{q} \rightarrow p = \frac{q}{q-1}$$

Получим:
$$y = x^{p-1} \rightarrow x = y^{\frac{1}{p-1}} = y^{\frac{1}{\frac{q}{q+1} -1}} = y^{q-1} = x$$




![[2.png]]


$ab \le S_1 + S_2$


$$S_1 = \int_0^a x^{p-1}dx = \frac{x^p}{p}\Bigg|_0^a = \frac{a^p}{p}$$
$$S_2 = \int_0^b y^{q-1}dx = \frac{y^q}{q}\Bigg|_0^b = \frac{b^q}{q}$$


Неравенство Гельдера:
$$\sum a_i b_i \le \left(\sum a_i^p \right)^{\frac{1}{p}} \left(\sum b_i^q \right)^{\frac{1}{q}}$$


Докажем это неравенство:
$$\frac{a_i}{\left(\sum a_i^p \right)^{\frac{1}{p}}} \cdot \frac{b_i}{\left(\sum b_i^q \right)^{\frac{1}{q}}} \le \frac{1}{p} \left(\frac{a_i}{\left(\sum a_i^p \right)^{\frac{1}{p}}} \right)^p \left(\frac{b_i}{\left(\sum b_i^q \right)^{\frac{1}{q}}} \right)^q = \frac{1}{p} \cdot \frac{a_i^p}{\sum a_i^p} + \frac{1}{q} \cdot \frac{b_i^q}{\sum b_i^q} \tag{**}$$

Просуммируем $(**)$  по $i$ от $1$ до $n:$
$$\frac{\sum a_i b_i}{\left(\sum a_i^p \right)^{\frac{1}{p}} \left(\sum b_i^q \right)^{\frac{1}{q}}} \le 1$$

### Конспект лекции №8: Геометрический подход — распознавание по расстояниям

**Главная идея:** Мы отказываемся от вероятностных моделей. Вместо этого мы постулируем, что объекты одного класса **близки** друг к другу в пространстве признаков, а объекты разных классов — **далеки**. Классификация сводится к сравнению расстояний от нового объекта до эталонов (типичных представителей) классов.

**Ключевая гипотеза — гипотеза компактности:**
> Схожие в метрическом смысле объекты гораздо чаще лежат в одном классе, чем в разных.

---

### Введение: Статистика vs. Геометрия

| Статистический подход | Геометрический (детерминистский) подход |
|----------------------|------------------------------------------|
| Знаем (или оцениваем) распределения | Не знаем никаких распределений |
| Используем вероятности и плотности | Используем расстояния |
| Решение = максимум апостериорной вероятности | Решение = минимум расстояния до эталона |
| Ошибка — вероятность | Ошибка — частость (доля ошибок на тесте) |
| Требует много данных для оценки плотности | Может работать с небольшими выборками |

**Важное замечание:** При определённых условиях (например, для нормальных распределений с одинаковыми диагональными ковариациями) эти подходы совпадают.

---

### Глава 1. Метрики и функции расстояния

Прежде чем сравнивать объекты, нужно определить, что значит "близко".

**Определение метрики** (строгое): Функция $d(x^{(i)}, x^{(j)})$ называется метрикой, если для любых $x^{(i)}, x^{(j)}, x^{(k)}$:
1. $d(x^{(i)}, x^{(i)}) = 0$ (расстояние от точки до себя = 0)
2. $d(x^{(i)}, x^{(j)}) = d(x^{(j)}, x^{(i)})$ (симметричность)
3. $d(x^{(i)}, x^{(j)}) = 0 \Rightarrow x^{(i)} \equiv x^{(j)}$ (только одинаковые точки дают ноль)
4. $d(x^{(i)}, x^{(j)}) \le d(x^{(i)}, x^{(k)}) + d(x^{(k)}, x^{(j)})$ (неравенство треугольника)

Если последние два свойства не выполняются, это просто **функция расстояния** (нестрогая мера).

#### Самые популярные функции расстояния

| Название | Формула | Особенность |
|----------|---------|-------------|
| **Евклидово** | $d_E = \sqrt{\sum_{s=1}^n (x_s^{(i)} - x_s^{(j)})^2}$ | Самое популярное, интуитивно понятно |
| **Манхэттенское** ($l_1$) | $d_1 = \sum_{s=1}^n |x_s^{(i)} - x_s^{(j)}|$ | Робастно к выбросам |
| **Минковского** ($l_p$) | $d_p = \left(\sum_{s=1}^n |x_s^{(i)} - x_s^{(j)}|^p\right)^{1/p}$ | Обобщает Евклида ($p=2$) и Манхэттен ($p=1$) |
| **Махаланобиса** | $d_M = \sqrt{(x^{(i)} - x^{(j)})^T C^{-1} (x^{(i)} - x^{(j)})}$ | Учитывает корреляцию признаков |

### Глава 2. Случай одного эталона на класс (простейший классификатор)

Пусть у нас есть $M$ классов, и каждый класс представлен **одним эталонным образом** $z^{(i)}$ (например, идеальное изображение буквы "А").

#### 2.1. Правило классификации

Новый образ $x$ относится к тому классу, до эталона которого расстояние минимально:
$$\omega_i : d(x, z^{(i)}) \le d(x, z^{(j)}) \quad \forall j \ne i$$

#### 2.2. Анализ для евклидова расстояния

Используем квадрат евклидова расстояния (монотонное преобразование, не меняет порядка):
$$D^{(i)} = (x - z^{(i)})^T (x - z^{(i)}) = x^T x - 2x^T z^{(i)} + z^{(i)T} z^{(i)}$$

Член $x^T x$ одинаков для всех классов. Поэтому минимизация расстояния эквивалентна **максимизации**:
$$g^{(i)}(x) = x^T z^{(i)} - \frac{1}{2} z^{(i)T} z^{(i)}$$

**Ключевой вывод:** Разделяющие функции $g^{(i)}(x)$ — **линейные** по $x$!

#### 2.3. Геометрическая интерпретация

Граница между классами $i$ и $j$ — это **гиперплоскость**:
$$x^T (z^{(i)} - z^{(j)}) + \frac{1}{2}(z^{(j)T} z^{(j)} - z^{(i)T} z^{(i)}) = 0$$

Свойства этой границы:
- Она перпендикулярна отрезку, соединяющему эталоны $z^{(i)}$ и $z^{(j)}$
- Проходит через середину этого отрезка
- Для $M > 2$ области решений — **выпуклые многогранники** (диаграмма Вороного)

**Пример из лекции (Рис. 1):**
- Для двух эталонов — одна прямая линия.
- Для четырёх эталонов — области решений образуют многоугольники.
- Для шести эталонов — диаграмма Вороного с шестью ячейками.

### Глава 3. Случай нескольких эталонов на класс

Часто один эталон не может представить всё разнообразие класса (например, буква "А" в разных шрифтах). Тогда каждый класс представляется множеством эталонов:
$$Z^{(i)} = \{z^{(i,1)}, z^{(i,2)}, ..., z^{(i,L)}\}$$

#### 3.1. Правило классификации

Расстояние от $x$ до класса $i$ — это **минимальное** расстояние до любого эталона этого класса:
$$d(x, Z^{(i)}) = \min_{k} d(x, z^{(i,k)})$$

Решение принимается в пользу класса с минимальным расстоянием:
$$\omega_i : d(x, Z^{(i)}) \le d(x, Z^{(j)}) \quad \forall j \ne i$$

#### 3.2. Эквивалентная форма

Разделяющая функция для класса $i$:
$$g^{(i)}(x) = \max_{k} \left[ x^T z^{(i,k)} - \frac{1}{2} z^{(i,k)T} z^{(i,k)} \right]$$

**Геометрически:** Область решений класса — это **объединение** областей Вороного для каждого его эталона.

#### 3.3. Связь с методом $k$-ближайших соседей

Это прямое обобщение правила одного ближайшего соседа (1-NN)!

| Метод | Правило |
|-------|---------|
| 1-NN (статистический) | Относим $x$ к классу ближайшего обучающего примера |
| Метод эталонов (1 эталон/класс) | Относим $x$ к классу ближайшего эталона |
| Метод эталонов (L эталонов/класс) | Относим $x$ к классу ближайшего эталона среди всех |

**Разница:** В $k$-NN мы храним **все** обучающие примеры. В методе эталонов — только отобранные (эталоны). Это даёт экономию памяти и скорости.

### Глава 4. Обучение метрических алгоритмов: как выбрать эталоны?

На практике эталоны часто неизвестны заранее. Есть только обучающая выборка $X^{N_i}$ для каждого класса. Нужно выбрать из неё **наиболее информативные** образы в качестве эталонов.

#### 4.1. Какие бывают образы в обучающей выборке?

| Тип | Характеристика | Что с ним делать? |
|-----|----------------|-------------------|
| **Типичные** | Находятся в центре класса | Обязательно оставить как эталоны |
| **Периферийные** | На краю класса, но внутри | Можно удалить — не сильно влияют |
| **Шумовые** | Попали в область другого класса (ошибка разметки) | Лучше удалить — портят качество |

#### 4.2. Алгоритм STOLP (Selective Training of Learning Patterns)

**Основная идея:** Выбрать минимальное множество эталонов, которое **безошибочно классифицирует всю обучающую выборку**.

**Условие:** Для каждой обучающей точки расстояние до ближайшего эталона **своего** класса должно быть **меньше**, чем до ближайшего эталона **чужого** класса.

**Сложность:** Задача комбинаторная (перебор всех подмножеств). STOLP — эвристический алгоритм, который находит хорошее (не обязательно оптимальное) решение.

#### 4.3. Более продвинутый вариант: FRiS–STOLP

Использует не просто расстояние, а **FRiS-функции** (Function of Rival Similarity) — меру схожести двух объектов **относительно** третьего.

Простыми словами: FRiS отвечает на вопрос "насколько $x$ похож на $z$ **по сравнению** с другим объектом $r$?" Это позволяет лучше учитывать "конкуренцию" между классами.

### Глава 5. Частный случай: бинарные признаки

Если признаки — только 0 или 1 (да/нет), обычные евклидовы расстояния работают, но есть и специализированные меры.

#### Популярные меры для бинарных векторов

| Название | Формула | Смысл |
|----------|---------|-------|
| **Нормированное скалярное произведение** | $d = \frac{x^T z}{\sqrt{(x^T x)(z^T z)}}$ | Косинус угла между векторами |
| **Мера Танимото** | $d = \frac{x^T z}{x^T x + z^T z - x^T z}$ | Отношение пересечения к объединению |

**Интерпретация для бинарных признаков:**
- $x^T z$ — количество признаков, где оба вектора имеют 1 (совпадение единиц)
- $x^T x$ — количество единиц в $x$
- $z^T z$ — количество единиц в $z$

Структура решающих правил остаётся той же — минимум расстояния до эталона.

### Итог лекции №8 (Что нужно вынести)

1. **Смена парадигмы:** Вместо вероятностей — расстояния. Вместо плотностей — эталоны.

2. **Гипотеза компактности** — фундамент подхода: близкие объекты скорее принадлежат одному классу.

3. **Классификатор по эталонам:**
   - 1 эталон/класс → линейные разделяющие функции, границы — гиперплоскости
   - L эталонов/класс → кусочно-линейные границы (объединение диаграмм Вороного)

4. **Связь со статистикой:** Метод эталонов — частный случай метода $k$-NN (при $k=1$ и отобранных эталонах вместо всей выборки).

5. **Обучение (выбор эталонов):**
   - Задача: из обучающей выборки выбрать минимальное множество эталонов
   - Алгоритмы: STOLP, FRiS–STOLP
   - Цель: убрать шум и периферию, оставить типичных представителей

6. **Для бинарных признаков** используются специальные меры (Танимото, косинусное расстояние).

7. **Преимущества метрического подхода:**
   - Простота интерпретации
   - Не требует знания распределений
   - Работает с малыми выборками
   - Быстрая классификация (особенно после отбора эталонов)

8. **Недостатки:**
   - Нужно уметь выбирать эталоны
   - Чувствителен к масштабу признаков (требует нормализации)
   - Не даёт оценку уверенности (в отличие от вероятности)

### Конспект лекции №9: Нелинейные преобразования, спрямляющие пространства и метод потенциальных функций

**Главная идея:** В реальном мире данные часто **линейно не разделимы** — никакая прямая (или гиперплоскость) не может разделить классы без ошибок. Решение: преобразовать данные в пространство **более высокой размерности**, где они станут линейно разделимыми. А затем — спроецировать решение обратно, получив в исходном пространстве **нелинейную границу**.

**Ключевая теорема — теорема Ковера:**
> Нелинейное преобразование сложной задачи классификации в пространство более высокой размерности повышает вероятность линейной разделимости образов.

Более того, для **любой** конечной обучающей выборки можно построить такое преобразование, которое сделает классы линейно разделимыми (ценой роста размерности).

---

### Глава 1. Проблема линейной неразделимости

#### 1.1. Классический пример: "Исключающее ИЛИ" (XOR)

Рассмотрим четыре точки на плоскости:

| $x_1$ | $x_2$ | Класс |
|-------|-------|-------|
| 0 | 0 | $\omega_1$ |
| 1 | 1 | $\omega_1$ |
| 1 | 0 | $\omega_2$ |
| 0 | 1 | $\omega_2$ |

**Визуализация:** Точки (0,0) и (1,1) — одного класса, (1,0) и (0,1) — другого. Никакая прямая не может разделить эти два класса. Это **линейно неразделимая задача**.

#### 1.2. Другой пример: "Заплетённые восьмёрки" (рис. 1б из лекции)

Два класса, каждый из которых состоит из двух несвязанных областей (эллипсов), расположенных "внахлёст". Степень "заплетённости" управляется параметром $d_m$:
- $d_m = 0$ — классы полностью сливаются
- $d_m$ большое — каждый класс распадается на два отдельных сгустка

Такая конфигурация также **линейно не разделима**.

### Глава 2. Идея спрямляющих пространств

**Основной приём:** Отобразить исходное пространство признаков $\mathbb{R}^n$ в пространство **более высокой размерности** $\mathbb{R}^m$ ($m > n$) с помощью нелинейного преобразования $\varphi$:
$$\varphi: \mathbb{R}^n \to \mathbb{R}^m, \quad n < m$$

В новом пространстве (называемом **спрямляющим**) образы становятся **линейно разделимыми**. Затем строится **линейный** классификатор в $\mathbb{R}^m$, который после обратного проецирования даёт в исходном пространстве **нелинейную** разделяющую границу.

**💎 Ключевой вывод:** Нелинейная граница в исходном пространстве = линейная граница в спрямляющем пространстве.

### Глава 3. Радиально-базисные функции (РБФ) для преобразования

Один из способов построить отображение $\varphi$ — использовать **радиально-базисные функции**.

#### 3.1. Определение

Радиально-базисная функция (РБФ) — это функция, зависящая только от расстояния до центра $c^{(j)}$:
$$\varphi_j(x) = \varphi(\|x - c^{(j)}\|) = \exp\left(-\frac{\|x - c^{(j)}\|^2}{2\sigma_j^2}\right)$$

Это **колоколообразная** функция: максимальна в центре $c^{(j)}$ и стремится к нулю при удалении от центра.

**Параметры:**
- $c^{(j)}$ — центр РБФ (обычно выбирают обучающие точки)
- $\sigma_j$ — ширина (масштаб) функции

#### 3.2. Применение к задаче XOR

Для задачи XOR выбираем центры РБФ в **каждой** обучающей точке:
$$c^{(1)} = (0,0)^T,\; c^{(2)} = (1,1)^T,\; c^{(3)} = (1,0)^T,\; c^{(4)} = (0,1)^T$$

Преобразование $\varphi: \mathbb{R}^2 \to \mathbb{R}^4$:
$$\varphi(x) = (\varphi_1(x), \varphi_2(x), \varphi_3(x), \varphi_4(x))^T$$

Если выбрать $\sigma$ достаточно малым, чтобы РБФ не пересекались, то:
- Для точки $(0,0)$: $\varphi_1 \approx 1$, остальные $\approx 0$
- Для точки $(1,1)$: $\varphi_2 \approx 1$, остальные $\approx 0$
- Для точки $(1,0)$: $\varphi_3 \approx 1$, остальные $\approx 0$
- Для точки $(0,1)$: $\varphi_4 \approx 1$, остальные $\approx 0$

Теперь в 4-мерном пространстве эти точки стали **линейно разделимыми**! Простой линейный классификатор с весами $w = (1, 1, -1, -1)^T$ даёт:
- Класс $\omega_1$: $y^T w \ge 1$
- Класс $\omega_2$: $y^T w \le -1$

#### 3.3. Проблема

Размерность спрямляющего пространства равна **числу обучающих точек** $N$. Если $N$ велико (тысячи, миллионы), работа в таком пространстве становится **вычислительно невозможной**.

Нужен способ работать в спрямляющем пространстве **неявно**, без явного вычисления $\varphi(x)$.

### Глава 4. Ядра скалярных произведений (kernel trick)

#### 4.1. Основная идея

Заметим, что почти все линейные классификаторы используют только **скалярные произведения** векторов:
- Для эталонного метода: $x^T z$
- Для линейной разделяющей функции: $w^T x$

В спрямляющем пространстве скалярное произведение имеет вид $\varphi(x)^T \varphi(z)$. Если мы сможем вычислить эту величину **без явного построения $\varphi(x)$**, мы сможем работать в спрямляющем пространстве **неявно**.

**Определение ядра:**
$$K(x, z) = \varphi(x)^T \varphi(z)$$

Функция $K(x, z)$ называется **ядром скалярного произведения**.

#### 4.2. Преимущества использования ядер

| Прямой подход | Ядерный подход |
|---------------|----------------|
| Явно вычисляем $\varphi(x)$ (размерность $m$) | Вычисляем $K(x, z)$ (скаляр) |
| Сложность $O(m)$ на одно скалярное произведение | Сложность $O(n)$ на одно вычисление ядра |
| Требует памяти для хранения $\varphi(x)$ | Не требует хранения $\varphi(x)$ |

Для многих ядер $m$ может быть **бесконечностью**, но $K(x, z)$ вычисляется за $O(n)$ операций!

#### 4.3. Расстояние через ядро

Важное свойство: можно выразить расстояние в спрямляющем пространстве через ядро:
$$d_E^2(\varphi(x), \varphi(z)) = K(x, x) - 2K(x, z) + K(z, z)$$

Таким образом, **метрические алгоритмы** (ближайшего соседа, эталонов) могут работать непосредственно с ядрами!

#### 4.4. Теорема Мерсера (критерий, что функция является ядром)

Функция $K(x, z)$ является ядром (т.е. существует $\varphi$, такое что $K(x, z) = \varphi(x)^T \varphi(z)$) тогда и только тогда, когда:
1. $K(x, z)$ **симметрична**: $K(x, z) = K(z, x)$
2. $K(x, z)$ **неотрицательно определена**: для любой конечной выборки $X^N = \{x^{(1)}, ..., x^{(N)}\}$ матрица $[K(x^{(i)}, x^{(j)})]$ неотрицательно определена (все собственные числа $\ge 0$)

#### 4.5. Популярные ядра (из лекции)

| Название | Формула | Особенность |
|----------|---------|-------------|
| **Гауссовское (RBF)** | $K(x, z) = \exp\left(-\frac{\|x - z\|^2}{2\sigma^2}\right)$ | Самое популярное, соответствует бесконечномерному пространству |
| **Полиномиальное** | $K(x, z) = (x^T z + 1)^p$ | Степень $p$ определяет сложность границы |
| **Сигмоидное** | $K(x, z) = \tanh(a x^T z + b)$ | Аналог нейронной сети |

### Глава 5. Метод потенциальных функций

Это классический метод, который исторически появился раньше, чем формализм ядер, но является его **частным случаем** (или предтечей). Он очень наглядно показывает, как "окружение" точек функциями приводит к линейной разделимости.

#### 5.1. Основная идея

Каждая точка обучающей выборки создаёт вокруг себя **потенциальное поле** (например, колокол). Значение потенциальной функции в точке $x$ от обучающей точки $x^{(i)}$:
$$K(x, x^{(i)}) = \exp\left(-\alpha \|x - x^{(i)}\|^2\right)$$

Суммарный потенциал в точке $x$ от всех точек класса $\omega_1$ и $\omega_2$:
$$K_1(x) = \sum_{x^{(i)} \in \omega_1} K(x, x^{(i)}), \quad K_2(x) = \sum_{x^{(j)} \in \omega_2} K(x, x^{(j)})$$

#### 5.2. Решающее правило

Объект $x$ относится к классу $\omega_1$, если:
$$K_1(x) > K_2(x)$$

То есть там, где "сильнее" потенциал от первого класса, мы относим точку к первому классу.

#### 5.3. Обучение (настройка весов)

В более общем виде потенциальная функция записывается как взвешенная сумма:
$$K(x) = \sum_{i=1}^{N} w_i K(x, x^{(i)})$$

Знак $K(x)$ определяет класс.

**Алгоритм обучения (итеративный):**
1. Начинаем с нулевых весов $w_i = 0$
2. Подаём обучающие точки по очереди
3. Если точка $x^{(i)}$ классифицируется правильно — ничего не меняем
4. Если точка $x^{(i)}$ классифицируется неправильно:
   - $w_i := w_i + 1$, если $x^{(i)} \in \omega_1$
   - $w_i := w_i - 1$, если $x^{(i)} \in \omega_2$
5. Повторяем проходы по выборке, пока все точки не будут классифицированы правильно (или пока не сойдётся)

**Теорема (Новикова):** Если данные линейно разделимы в пространстве, порождённом ядром, алгоритм сойдётся за конечное число итераций.

#### 5.4. Связь с методом опорных векторов (SVM)

Метод потенциальных функций — это предшественник **метода опорных векторов (SVM)**. Отличие:
- В методе потенциальных функций веса $w_i$ могут быть любыми целыми числами
- В SVM веса $w_i$ отличны от нуля только для **опорных векторов** (тех точек, которые лежат на границе), и их значения определяются решением задачи квадратичного программирования

#### 5.5. Преимущества и недостатки метода потенциальных функций

| Преимущества | Недостатки |
|--------------|-------------|
| Простая итеративная процедура обучения | Может не сойтись, если данные неразделимы |
| Не требует задания размерности спрямляющего пространства | При большом $N$ — много вычислений |
| Хорошо интерпретируется ("потенциалы" от точек) | Выбор параметра $\alpha$ (ширины) критичен |

### Глава 6. Обобщённая структура

Всё, что мы обсудили, можно представить как общую схему:

```
Исходные признаки x ∈ ℝⁿ
        ↓
Ядерное отображение (неявное) через K(x, z)
        ↓
Линейный классификатор в спрямляющем пространстве
        ↓
Решающее правило: sign( Σ w_i K(x, x^{(i)}) + w₀ )
        ↓
(Опционально) Вычисление расстояний и метод k-NN с ядром
```

**Фактически:** любой линейный метод, переписанный через скалярные произведения, можно "ядерно расширить", заменив $x^T z$ на $K(x, z)$, и получить нелинейный аналог.

### Итог лекции №9 + метод потенциальных функций (Что нужно вынести)

1. **Проблема:** Реальные данные часто линейно не разделимы (XOR, заплетённые восьмёрки).

2. **Решение:** Отобразить данные в пространство **более высокой размерности** (спрямляющее пространство), где они становятся линейно разделимыми. Это гарантируется теоремой Ковера.

3. **Радиально-базисные функции (РБФ):** Один из способов явного построения отображения. Каждая точка "окружается" колоколом. Недостаток: размерность растёт с числом точек.

4. **Ядра скалярных произведений (kernel trick):** Позволяют работать в спрямляющем пространстве **неявно**, вычисляя только скалярные произведения через ядро $K(x, z)$. Это открыло дорогу SVM и другим методам.

5. **Метод потенциальных функций:** Исторически первый метод, использующий эту идею. Обучается простым итеративным алгоритмом: ошиблись — добавили/вычли вес.

6. **Популярные ядра:**
   - Гауссовское (RBF) — самое универсальное
   - Полиномиальное — задаёт границу в виде полинома
   - Сигмоидное — связано с нейронными сетями

7. **Ключевая формула (расстояние через ядро):**
   $$d^2(x, z) = K(x, x) - 2K(x, z) + K(z, z)$$

8. **Главный практический вывод:** Если ваш линейный классификатор плохо работает, не усложняйте его — **примените ядерный трюк** к уже существующему линейному методу. Это часто даёт значительное улучшение качества.

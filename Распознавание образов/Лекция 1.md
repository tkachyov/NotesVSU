## Обработка информации, анализ данных, машинное обучение. Основные понятия и определения

1. Базовые понятия: Данные, Информация, Сигнал
   - Данные:
     * Первичные, необработанные результаты измерений и наблюдений.
     * Зафиксированы на материальном носителе.
     * Являются "сырьем" для последующего анализа.
   - Информация:
     * Результат анализа и обработки данных.
     * "Вторичные", преобразованные данные, пригодные для принятия решений.
     * Обладают смысловым содержанием и уровнем обобщения.
   - Сигнал:
     * Физический носитель (электромагнитный, акустический и т.д.) данных и информации.
     * Используется для передачи от объекта к потребителю.

2. Системы и процессы обработки
   - Обработка информации (ОИ): Систематическое выполнение операций над данными (ГОСТ 15971-90).
   - Информационная система (ИС): Человеко-машинная система, предназначенная для преобразования данных и информации в интересах пользователей.
   - Информационный процесс: Организованная последовательность операций, изменяющих форму и/или содержание информации в пространстве и времени.
   - Информационная технология: Совокупность приемов и способов (технических, программных, сетевых) для реализации информационного процесса.
   - Технология обработки информации: Совокупность приемов и способов для извлечения информации об объектах при взаимодействии с ними.
   - Операции преобразования: Элементарные действия в рамках процесса.
   - Алгоритм: Инструкция, определяющая последовательность действий для достижения результата.

3. Система обработки информации (СОИ)
   - Определение: Совокупность технических и программных средств, методов и персонала для автоматизированной обработки информации.
   - Целевое назначение: Анализ и обобщение больших объемов данных для получения новой информации о прошлом, текущем или будущем состоянии объекта, интересующей потребителя.
   - Примеры: Системы технической диагностики, OLAP-системы, системы аналитической отчетности.

4. Анализ данных и смежные понятия
   - Анализ данных (АД):
     * Наука об извлечении знаний из данных.
     * Процесс исследования, фильтрации, моделирования данных для извлечения полезной информации.
   - Искусственный интеллект (ИИ): Междисциплинарное направление по созданию систем, способных накапливать знания и применять их подобно человеку.
   - Data Mining: Одно из прикладных направлений, тесно связанное с анализом данных (например, поиск ассоциативных правил).

5. Наше понимание терминологии (Иерархия понятий)
   - Обработка информации: Наиболее широкое понятие, охватывающее все типы преобразований данных.
   -  Анализ данных: Более узкое понятие, нацеленное на интеллектуальную обработку для извлечения знаний и принятия решений.
   - Машинное обучение (ML): Базовый подход в анализе данных, основанный на построении математических моделей и синтезе алгоритмов, способных к самообучению.
      * Важное замечание: Анализ данных возможен и без машинного обучения, если модель объекта задана (постулирована) заранее.

6. Классификация задач анализа данных
   - По цели:
     * Описательные (Descriptive): Понимание данных (кластеризация, ассоциация).
     * Предсказательные (Predictive): Предсказание состояний объектов (классификация, регрессия, оценивание).
   - По типу обучения:
     * С учителем (Supervised learning): Используются размеченные данные (задачи классификации, регрессии).
     * Без учителя (Unsupervised learning): Данные не размечены (задачи кластеризации, поиска ассоциаций).

7. Терминология для постановки задач
   - Наблюдения (z ∈ R^q): Первичные данные с датчиков.
   - Признаки / Дескрипторы (x ∈ R^n): Отобранные или преобразованные наблюдения, используемые для описания объекта. Обычно n < q.
   - Состояние (s ∈ R^m): Ненаблюдаемые напрямую параметры объекта, которые нужно оценить.
   - Образ (x): Формализованное описание объекта в пространстве признаков.
   - Класс образов (Ω = {ω1, ..., ωM}): Совокупность схожих объектов.
   - Эталонное описание класса: Математическая модель (априорные характеристики) класса.
   - Обучение: Процедура получения эталонных описаний на основе обучающих данных.
   - Принятие решения: Процедура отнесения объекта к классу или оценки его параметров на основе модели.
   - Решающее правило: Алгоритм, лежащий в основе принятия решения.

8. Обучающая и тестирующая выборки
   - Обучающая выборка (X^N): Набор прецедентов (векторов признаков) для обучения.
     * Для обучения с учителем дополнительно задается:
       + D^N (метки классов) или
       + Y^N (вектор "ответов" для регрессии).
     * Для обучения без учителя задается только X^N.
   - Тестирующая выборка: Набор данных для проверки качества обученного алгоритма. Обязательно должна быть размечена (метки или эталонные ответы) для объективной оценки.

9. Детальный разбор типовых задач
   - Задача 1: Распознавание (Классификация)
     * Отнесение объекта к одному из известных классов.
     * Классы и их эталонные описания либо известны априори, либо определяются в процессе обучения.
   - Задача 2: Кластеризация
     * Разбиение множества объектов на группы (кластеры) схожих между собой.
     * Число кластеров может быть как известным, так и неизвестным (автоматическая классификация).
   - Задача 3: Оценивание
     * Определение неизвестных характеристик (параметров состояния s) объекта.
     * Параметрическое оценивание: Оценка параметров известного распределения.
     * Непараметрическое оценивание: Оценка вида распределения.
     * Фильтрация: Оценка ненаблюдаемой реализации случайного процесса (сигнала).
   - Задача 4: Регрессия
     * Установление функциональной зависимости между входными (независимыми) и выходными (зависимыми) переменными.
     * Используется для предсказания.
     * Прогнозирование: Регрессия, где независимая переменная — время.
   - Задача 5: Ассоциация
     * Поиск значимых связей (правил) между объектами или событиями (например, в Data Mining). Часто рассматривается как частный случай классификации.

10. Основные подходы к анализу данных
    - 1. Статистический подход:
      * Использует вероятностные модели (функции распределения, моменты).
      * Качество оценивается вероятностными характеристиками (риски, дисперсии ошибок).
      * Методы делятся на байесовские и небайесовские.
    - 2. Детерминистский подход:
      * Отказ от вероятностных моделей.
      * Метрический подход: Использует функции расстояния (метрики) и гипотезы о компактности данных.
      * Нейросетевой подход: Представление алгоритма в виде универсального преобразователя "вход-выход" (нейронной сети), требующего обучения.
      * Структурно-лингвистический подход: Использование непроизводных элементов (подобразов) и грамматики для описания образов, что позволяет применять теорию формальных языков.

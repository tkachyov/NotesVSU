### Немарковские СМО.

#### Метод вложенных цепей Маркова.

Анализ немарковских СМО в случае, если марковское свойство в отсутствие действия проявляется в некоторый момент времени основан на методе вложенных цепей Маркова.

Примем, что найдётся последовательность, случайно детерминированная, что в момент $t_n$ отсутствует после действия, то есть будущая система зависит от текущего состояния и свойств потоков в будущем.
$$\exists \{t_n \}$$
$$P(\lim_{n \rightarrow \infty} t_n = \infty) = 1$$$\nu$ - число заявок, находящихся в системе.

$$\lim_{n \rightarrow \infty} P(\nu(t_n) = m) = P_m$$
то есть устанавливается распределение для специальных моментов времени.

$$\lim_{n \rightarrow \infty} P(\nu(t) = m) = P_m$$


В качестве иллюстрации метода влияния цепей Маркова рассмотрим одноканальные СМО с неограниченной очередью,  на вход которой поступает простейший поток интенсивности $\lambda$.
А время обслуживания имеет произвольное распределение заданной функции $H(t) = P(\mu_k < t).$ $\mu_k$ - время обслуживания $k$-й заявки, поступающей в систему.


$$f(t) \longrightarrow \hat{f} (s) = \int_{0}^{+ \infty} f(t) e^{-st} dt$$

$\{t_n \}$- последовательность момента времени, соответствующего окончанию обслуживания $n$-той заявки.


$\nu (t_n + 0) = \nu_n$ - число заявок в системе в момент времени $t_n$ после того, как $n$-тая заявка покинула систему.

$\nu_n$ обладает марковским свойством:
Если в момент времени система освобождается, то начинается процесс ожидания $n+1$ заявки (в силу свойств простейшего потока обладает марковским свойством), либо начинается обслуживание $n+1$ заявки. В любом случае будущего процесса не зависит от прошлого.

$\triangle_n$  - число заявок, пришедших в систему за время обслуживания $n$-той заявки.

$$
\begin{equation*} 
\nu_n = 
\begin{cases} 
\nu_{n-1} + \triangle_n -1, \nu_{n-1} > 0 \\ 
\triangle_n, \nu_{n-1} = 0 
\end{cases} 
\end{equation*}
$$

$$P_{ij}^{(n)} = P (\nu_n = j | \nu_{n-1} = i)$$


Рассмотрим переходные вероятности:

$$P(A) = \sum_{i=1}^n P(H_i) P(A|H_i)$$- это формула полной вероятности.


$$P_{ij}^{(n)} = \int_0^{\infty} \frac{(\lambda x)^j}{j!} e^{- \lambda x} dH (x)$$
это для $i = 0,1$ и $j = 0, 1, 2, ...$


$$P_{ij}^{(n)} = \int_0^{\infty} \frac{(\lambda x)^{j+1-i}}{(j+1-i)!} e^{- \lambda x} dH (x)$$
это для $j > i-1$ и $i \ge 2$.


$\frac{(\lambda t)^n e^{- \lambda t}}{n!}$ - закон распределения Пуассона.


$f_k = \int_0^{+ \infty} \frac{e^{- \lambda x} (\lambda k)^k}{k!} d H(x)$

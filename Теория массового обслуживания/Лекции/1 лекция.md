Система, призванная многократно выполнять однотипные операции называется системой массового обслуживания (СМО).

Особенностью СМО является, что заявки поступают в случайные моменты времени, и время обслуживания также является случайной величиной.

$\{t_n\}$ - случайная последовательность (случайный поток заявок), где индекс $n$ будет соответствовать моменту времени $n$-той по счёту заявки.


$P \{t_n - t_{n-1} \} = 1$, то есть $P \{t_n < x \} \underset{n \to \infty}{\longrightarrow} 0$.
Соотношение означает, что за любое конечное время в системе не может быть $\infty$ число заявок.



#### Показательное распределение и его свойства

Пусть случайная величина $\xi \sim exp(\lambda)$.

$$
f_{\xi} = 
\begin{cases} 
0, & \text{если } x < \infty \\
\lambda e^{- \lambda x}, & \text{если } x \ge 0 
\end{cases}
$$
![[1.jpg]]


$$M \xi = \int_{- \infty}^{+ \infty} x f(x)dx = \int_{0}^{- \infty} \lambda x e^{-\lambda x} dx = \left\{ \begin{cases} 
u = \lambda x, & du = \lambda dx \\
dv = e^{- \lambda x}, & v = -\frac{e^{- \lambda x}}{\lambda} 
\end{cases} \right\} = $$ $$= x e^{- \lambda x} \Big|_0^{\infty} + \int e^{- \lambda x} dx = -\frac{1}{\lambda} e^{- \lambda x} \Big|_0^{\infty} =\frac{1}{\lambda}$$


$$F(x) = \int_{- \infty}^x f_{\xi} (s) ds$$

$$F(x) = P \{\xi < x \} = 
\begin{cases} 
0, x < 0 \\
\int_0^x \lambda e^{-\lambda s} = - e ^{-\lambda s} \Big|_0^x = 1 - e ^{-\lambda x} 
\end{cases}
$$
![[2.jpg]]


$P \{\xi > x \} = 1 - (1 - e^{\ \lambda x}) = e^{- \lambda x}$ - следовательно, $\xi$ - показательная случайная величина с параметром $\lambda$.


##### Свойства

1. $P \{\xi < h \} = \lambda h + o(h)$
	Доказательство:$$P \{\xi < h \} = 1 - e^{- \lambda h} = 1 - (1 - \lambda h + o(h)) = \lambda h + o(h)$$
2. $P \{ \xi > h \} = 1 - \lambda h + o(h)$
3. $P \{\xi > x + t | \xi  > x\} =e^{-\lambda t}$	$$\xi > x + t = A; \xi  > x = B$$
$$P(A|B) = \frac{P(AB)}{P(B)}$$
$$P(AB) = P(B)P(A|B)$$
$$P(AB) = \frac{P \{(\xi > x + t) \cdot (\xi > x)\}}{P \{\xi > x \}} = \frac{P \{\xi > x + t \}}{P \{\xi > x \}} = \frac{e^{-\lambda (x+t)}}{e^{- \lambda x}} = e^{- \lambda t}$$

Таким образом, доказанное соотношение означает следующее: если рассматривается некоторая операция, продолжительность которой есть показательная распределённая случайная величина, то если эта операция не завершилась в течение некоторого (произвольного) времени, то остаток времени будет иметь исходное распределение с тем же распределением $\lambda$.


4. Пусть $\xi_1, \xi_2, ..., \xi_n \sim exp (\lambda)$.
	$\gamma_i (i = 1, 2, ..., n)$ - операция длительностью $\lambda_{\xi}$
	$\gamma_i \sim \xi_i$
	$\gamma = \underset{i}{min} \{\gamma_i \}$
	$\gamma \sim exp(\lambda)$
	$\lambda = \lambda_1 + \lambda_2 + ... + \lambda_n$


$$P \{\gamma > x \} = P \{\underset{i}{min} \gamma_i > x\} = P \{(\gamma_1 > x)(\gamma_2 > x)... (\gamma_n > x)\} = P(\gamma_1 > x)P(\gamma_2 > x)...P(\gamma_n > x)=$$
$$= e^{-\lambda_1 x} e^{-\lambda_2 x}...e^{-\lambda_n x} = e^{-(\lambda_1 + \lambda_2 + ...+\lambda_n )x} = e^{-\lambda x}$$

1) $P \{...\} = \frac{\lambda_1}{\lambda}$ 
(вероятность того, что первой закончится операция $\gamma_i$) - доказать (Доказано)


Пусть $\gamma_1, \gamma_2, \dots, \gamma_n$ — независимые случайные величины,  
$\gamma_i \sim \mathrm{Exp}(\lambda_i)$.  
  
Обозначим $\gamma = \min\{\gamma_1, \dots, \gamma_n\}$.  
Требуется найти:  
$$
P\{\gamma_i = \gamma\}.  
$$
  
Имеем:  
$$
P\{\gamma_i = \gamma\} = P\{\gamma_i < \gamma_j, \ \forall j \ne i\}.  
$$  
Используем формулу полной вероятности:  
$$
P\{\gamma_i < \gamma_j, \ \forall j \ne i\}  
= \int_0^{\infty} P\{\gamma_i \in dx\} \cdot P\{\gamma_j > x, \ \forall j \ne i\}.$$
  
Так как случайные величины независимы, получаем:  
$$
= \int_0^{\infty} \lambda_i e^{-\lambda_i x} \cdot \prod_{j \ne i} e^{-\lambda_j x} \, dx.  
$$
$$
= \int_0^{\infty} \lambda_i e^{-(\lambda_1 + \dots + \lambda_n)x} \, dx.  
$$

Обозначим $\lambda = \sum_{k=1}^n \lambda_k$, тогда:  
$$
= \lambda_i \int_0^{\infty} e^{-\lambda x} \, dx = \lambda_i \cdot \frac{1}{\lambda}.  
$$  
Итак,  
$$
P\{\gamma_i = \gamma\} = \frac{\lambda_i}{\lambda}.  
$$



2) $P \{(\xi_i < x) (\xi_j (j \ne i) > x) \}$ - доказать (Доказано)

Пусть $\xi_i \sim \mathrm{Exp}(\lambda_i)$ и $\xi_j \sim \mathrm{Exp}(\lambda_j)$ —  
независимые случайные величины.  
  
Требуется найти:  
$$  
P\{(\xi_i < x)(\xi_j > x)\}.  
$$
Так как случайные величины независимы, имеем:  
$$
P\{(\xi_i < x)(\xi_j > x)\}  
= P\{\xi_i < x\} \cdot P\{\xi_j > x\}.  
$$  
Используя свойства показательного распределения:  
$$
P\{\xi_i < x\} = 1 - e^{-\lambda_i x}, \quad  
P\{\xi_j > x\} = e^{-\lambda_j x}.  
$$  
Следовательно,  
$$
P\{(\xi_i < x)(\xi_j > x)\}  
= (1 - e^{-\lambda_i x}) \cdot e^{-\lambda_j x}.  
$$






Простейший поток однородных событий следующей последовательности $\{t_n \}$ называется простым потоком однородных событий, если разница между двумя событиями есть показательно распределённая случайная величина с показателем $\lambda$.
$$t_n - t_{n-1} = \xi \sim exp(\lambda)$$
![[NotesVSU/Теория массового обслуживания/Файлы/3.jpg]]

$t_n = \xi_1 + \xi_2 + ... + \xi_n$


Закон распространения $n$-го события простейшего потока.
$$t_1 = \xi_1 \sim exp(\lambda)$$
$$t_2 = \xi_1 + \xi_2$$

$$f_{t_2}(x) = \int_{-\infty}^{+\infty} f_{\xi_1} (x-s) f_{\xi_2} (s) ds = $$ - свёртка функций

$$ = \int_0^x \lambda e^{-\lambda (x-s)} \lambda e^{-\lambda s} ds = \lambda^2 e^{-\lambda x} \int_0^x ds = \lambda^2 e^{-\lambda x} x$$
$$t_2 \sim f_{t_2} = \begin{cases} 
\lambda^2 e^{-\lambda x} x, & \text{если } x > 0 \\
0, & \text{если } x < 0 
\end{cases}$$
$$t_3 \sim \int_0^x x e^{- \lambda (x-s)} \lambda^2 e^{- \lambda s} s ds = \lambda^3 e^{- \lambda x} \int_0^x s ds = \frac{\lambda^3 e^{- \lambda x} x^2}{2}$$
$$t_4 \sim \frac{\lambda^4 e^{- \lambda x} x^3}{3!}$$
$$t_n \sim \frac{\lambda^n e^{- \lambda x} x^{n-1}}{(n-1)!}$$
-доказать по индукции!!! (Доказано!)


Пусть $\xi_1, \xi_2, \dots, \xi_n$ — независимые случайные величины,
$\xi_i \sim \mathrm{Exp}(\lambda)$.

Обозначим:
$t_n = \xi_1 + \xi_2 + \dots + \xi_n.$

Требуется доказать, что плотность распределения $t_n$ имеет вид:
$$f_{t_n}(x) = 
\begin{cases}
\frac{\lambda^n x^{n-1} e^{-\lambda x}}{(n-1)!}, & x > 0, \\
0, & x \le 0.
\end{cases}$$

База индукции: при $n = 1$ имеем:
$t_1 = \xi_1, \quad f_{t_1}(x) = \lambda e^{-\lambda x},$
что совпадает с формулой при $n = 1$.

Индукционное предположение: пусть для $n-1$ верно:
$f_{t_{n-1}}(x) = \frac{\lambda^{n-1} x^{n-2} e^{-\lambda x}}{(n-2)!}, \quad x > 0.$

Переход:

Так как
$t_n = t_{n-1} + \xi_n,$
то плотность $f_{t_n}(x)$ равна свёртке:
$f_{t_n}(x) = \int_0^x f_{t_{n-1}}(x - s)\, f_{\xi_n}(s)\, ds.$

Подставляя выражения для плотностей, получаем:
$$f_{t_n}(x) = \int_0^x 
\frac{\lambda^{n-1} (x - s)^{n-2} e^{-\lambda (x - s)}}{(n-2)!}
\cdot \lambda e^{-\lambda s} \, ds.$$

$$= \frac{\lambda^n e^{-\lambda x}}{(n-2)!}
\int_0^x (x - s)^{n-2} \, ds.$$

Сделаем замену $u = x - s$:
$$= \frac{\lambda^n e^{-\lambda x}}{(n-2)!}
\int_0^x u^{n-2} \, du.$$

$$= \frac{\lambda^n e^{-\lambda x}}{(n-2)!}
\cdot \frac{x^{n-1}}{n-1}.$$

$$= \frac{\lambda^n x^{n-1} e^{-\lambda x}}{(n-1)!}.$$

Следовательно, формула верна для $n$, а значит, по принципу математической индукции она справедлива для всех $n \in \mathbb{N}$.





#### Распределение числа событий простейшего потока на конечном временном интервале


Рассмотрим отрезок $[0, h], h \to 0$

$\nu_n$ - число событий простейшего потока на интервале.

1) $P \{\nu_n = 0 \}  = 1 - \lambda h + o(h)$
2) $P \{\nu_n = 1 \}  = \lambda h + o(h)$
3) $P \{\nu_n \ge 1 \}  = \lambda h + o(h)$
4) $P \{\nu_n \ge 2 \}  = o(h)$

Доказательство:
1) $P \{\nu_n = 0 \}  \equiv P \{\xi_1 > h\} = e^{-\lambda h} = 1 - \lambda h + o(h)$
2) $P \{\nu_n = 1 \}  = P \{(t_1 < h) (t_2 > h) \} = P(t_1 < h) P\{(t_2 > h) | t_1 < h\} = (\lambda h + o(h)) (1 - \lambda (h - t_1) + o(h - t_1)) = \lambda h - o(h)$
$$\{\nu_n \ge 1\} = \{\nu_n = 1 \} + \{\nu_n \ge 2 \}$$
$$\lambda h + o(h) = \lambda h + o(h) + P(\nu_n \ge 2) \Rightarrow P(\nu_n \ge 2)  = o(h)$$
3) $P(\nu_n \ge 1) = 1 - P(\nu_n = 0) = 1 - (1 - \lambda h + o(h)) = \lambda h + o(h)$

Таким образом, вероятностью появления двух и более событий простого потока на малом временном интервале можно пренебречь по сравнению с вероятностью появления одного события, это свойство называется *ординарностью*.

Из свойств показательного распределения следует, что вероятностные характеристики простейшего потока не меняется со временем:
1) Во всякий момент времени ожидание следующего события равно $\frac{1}{\lambda}$ и не зависит от прошлого.
2) Простейший поток обладает отсутствием после действия: количество заявок и момент их появления в прошлом времени никак не влияют на вероятностные характеристики в будущем.

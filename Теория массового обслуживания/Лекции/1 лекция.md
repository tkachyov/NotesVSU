Система, призванная многократно выполнять однотипные операции называется системой массового обслуживания (СМО).

Особенностью СМО является, что заявки поступают в случайные моменты времени, и время обслуживания также является случайной величиной.

$\{t_n\}$ - случайная последовательность (случайный поток заявок), где индекс $n$ будет соответствовать моменту времени $n$-той по счёту заявки.


$P \{t_n - t_{n-1} \} = 1$, то есть $P \{t_n < x \} \underset{n \to \infty}{\longrightarrow} 0$.
Соотношение означает, что за любое конечное время в системе не может быть $\infty$ число заявок.



#### Показательное распределение и его свойства

Пусть случайная величина $\xi \sim exp(\lambda)$.

$$
f_{\xi} = 
\begin{cases} 
0, & \text{если } x < \infty \\
\lambda e^{- \lambda x}, & \text{если } x \ge 0 
\end{cases}
$$
![[1.jpg]]


$$M \xi = \int_{- \infty}^{+ \infty} x f(x)dx = \int_{0}^{- \infty} \lambda x e^{-\lambda x} dx = \left\{ \begin{cases} 
u = \lambda x, & du = \lambda dx \\
dv = e^{- \lambda x}, & v = -\frac{e^{- \lambda x}}{\lambda} 
\end{cases} \right\} = $$ $$= x e^{- \lambda x} \Big|_0^{\infty} + \int e^{- \lambda x} dx = -\frac{1}{\lambda} e^{- \lambda x} \Big|_0^{\infty} =\frac{1}{\lambda}$$


$$F(x) = \int_{- \infty}^x f_{\xi} (s) ds$$

$$F(x) = P \{\xi < x \} = 
\begin{cases} 
0, x < 0 \\
\int_0^x \lambda e^{-\lambda s} = - e ^{-\lambda s} \Big|_0^x = 1 - e ^{-\lambda x} 
\end{cases}
$$
![[2.jpg]]


$P \{\xi > x \} = 1 - (1 - e^{\ \lambda x}) = e^{\ \lambda x}$ - следовательно, $\xi$ - показательная случайная величина с параметром $\lambda$.


##### Свойства

1. $P \{\xi < h \} = \lambda h + o(h)$
	Доказательство:$$P \{\xi < h \} = 1 - e^{- \lambda h} = 1 - (1 - \lambda h + o(h)) = \lambda h + o(h)$$
2. $P \{ \xi > h \} = 1 - \lambda h + o(h)$
3. $P \{\xi > x + t | \xi  > x\} =e^{\lambda t}$

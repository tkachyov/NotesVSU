**Пример.**

На многоканальный телефон поступает 30 звонков в час. Средняя продолжительность разговора t = 3 мин. При каком наименьшем количестве каналов обслуживания относительная способность будет не менее чем 0,9?

![[10.jpg]]


$$P_0 = (1 + \rho + \frac{\rho^2}{2} + ... + \frac{\rho^n}{n!})^{-1}$$

$$P_k = P_n \cdot \frac{\rho^2}{k!}, (k = 1, 2, ..., n)$$
$$P_{отк} = P_n = P_0 \frac{\rho^n}{n!}$$

$$Q = 1 - P_{отк} = 1 - P_0 \frac{\rho^n}{n!} > 0,9$$
$$\rho = \frac{\lambda}{\mu}$$
$$\mu = \frac{1}{t_{обс}}$$

$\rho = 30 \cdot \frac{1}{20} = 1,5$ - приведённая интенсивность потока заявок.

$$P_0 (n=3) = (1 + 1,5 + \frac{1,5^2}{2} + \frac{1,5^3}{6})^{-1} \approx 0,24$$
$$P_{отк} (n=3) = 0,24 \frac{1,5^3}{6} = 0,135 > 0,1$$
$$P_0 (n=4) = (1 + 1,5 + \frac{1,5^2}{2} + \frac{1,5^3}{6} + \frac{1,5^4}{24})^{-1} \approx 0,23$$
$$P_{отк} (n=4) = 0,24 \frac{1,5^4}{24} = 0,05$$
Ответ: при $n=4$. (При каком наименьшем количестве каналов обслуживания относительная способность будет не менее чем 0,9?)



$$Q = 0,95$$
$$A = \lambda Q = 30 \cdot 0,95 = 28,5$$
$$k = \frac{A}{\mu} = \frac{28,5}{20} = 1,4$$
1,4 среднее количество занятых каналоводновременно (в нашем примере на практике будет 2).

##### Одноканальные СМО с неограниченной очередью.

Рассмотрим систему, на вход которой поступает простейший поток заявок интенсивности $\lambda$. И есть единственный поток обслуживания $\mu$. Состояние системы будем характеризовать числом заявок в ней.

![[11.jpg]]


$$P_0 = (1 +\rho + \rho^2 + ... + \rho^n + ...)^{-1}$$
ряд сходится, когда $\rho < 1$.

$$\rho = \frac{\lambda}{\mu}$$
$$\rho < 1 \Leftrightarrow \lambda < \mu$$
$$P_0 = (\frac{1}{1 - \rho})^{-1} = 1 - \rho$$
$$P_n = P_0 \rho^n = (1 - \rho) \rho^n$$
при $\lambda \ge \mu$ очередь $\to \infty$. 


$P_{отк} = 0$, так как $\rho < 1$
$Q = 1$
$A = \lambda$

$T_{сист}$ - Среднее время пребывания заявки в системе.

$T_{оч}$ - Среднее время обслуживания очереди.

$L_{сист}$ - Среднее число заявок в системе.

$L_{оч}$ - Среднее число заявок в очереди.


$$L_{сист} = \sum_{n=1}^{\infty} n p_n = \sum_{n=1}^{\infty} (1 - \rho) n \rho^n = (1 - \rho) \rho \sum_{n=1}^{\infty} n \rho^{n-1}$$

$$\rho  (1 - \rho) \sum_{n=1}^{\infty} (x^n) \Bigg|_{x = \rho} = \rho (1 - \rho) (\frac{x}{1-x})' \Bigg|_{x = \rho} =  \rho (1 - \rho) \frac{1(1-x)-(-1)x}{(1-x)^2} \Bigg|_{x = \rho} = $$
$$= \rho  (1 - \rho) \frac{1}{(1 - \rho)^2} = \frac{\rho}{1 - \rho}$$


Таким образом, 
$$L_{сист} = \frac{\rho}{1 - \rho}$$
$$L_{сист} = L_{обс} + L_{оч}$$
$$L_{обс} = 0 \cdot P_0 + 1 (1 - P_0) = 1 - P_0 = 1 - (1 - \rho) = \rho$$
$$L_{оч} = L_{сист} - L_{оч} = \frac{\rho}{1 - \rho} - \rho = \frac{\rho^2}{1 - \rho}$$
$$T_{сист} = \frac{L_{сист}}{\mu}$$
$\mu$ - количество обслуженных заявок за единицу времени.


$$T_{оч} = \frac{L_{оч}}{\mu}$$


Система имеет n каналов обслуживания и неограниченное количество мест в очереди.
На вход поступает поток заявок интенсивности $\lambda$.
Интенсивность потока обслуживания равна $\mu$.
Состояние системы будем характеризовать числом заявок в ней.

![[12.jpg]]


$$P_0 = (1 + \rho + \frac{\rho^2}{2} + ... + \frac{\rho^n}{n!} + \frac{\rho^{n+1}}{n \cdot n!} +...+ \frac{\rho^{n+k}}{n! n^k})^{-1} = $$
$$= (1 + \rho + ... + \frac{\rho^n}{n!} + \frac{\rho^n}{n!}(\frac{\rho}{n} + ... + (\frac{\rho}{n})^k))^{-1} = $$
условие сходимости ряда: $\rho < 1$.

Состояние системы будет иметь ненулевые предельные вероятности при условии, что суммарная интенсивность потока обслуживания будет превосходить интенсивность потока заявок.

$$ = (1 + \rho + ... + \frac{\rho^n}{n!} + \frac{\rho^n}{n!}(\frac{\rho / n}{1 - \rho / n}))^{-1} = (1 + \rho + \frac{\rho^2}{2} + ... + \frac{\rho^n}{n!} + \frac{\rho^{n+1}}{n! (n - \rho)})^{-1}$$

$Q = 1$, $P_{отк} = 0$.

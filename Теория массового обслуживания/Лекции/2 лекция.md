#### Системы дифференциальных уравнений Колмогорова. Предельные вероятности. Теорема Колмогорова.

**Пример.**

Рассмотрим систему, состоящую из двух приборов, время работы каждого из которых  есть случайная величина, распределённая по показательному закону. В момент поломки каждого из приборов немедленно начинается его исправление, также продолжающееся случайное время, распределённое по показательному закону.

$$p_0(t) + p_1(t) + p_2(t) + p_3(t) \equiv 1$$
Это марковские цепи.


Выберем маленький промежуток времени $\Delta t$.
$$p_0 (t + \Delta t)$$

$\Delta t$ настолько мало, что переходом в состояния можно пренебречь.


I.
$$p_0 (t) (1 - (\lambda_{01} + \lambda_{02}) \Delta t)$$
$$p_0 (t + \Delta t) = p_0 (t) (1 - (\lambda_{01} + \lambda_{02}) \Delta t) + \lambda_{10} p_1(t) \Delta t + \lambda_{20} p_2(t) \Delta t + o(\Delta t)$$
$$\frac{p_1 (1 + \Delta t) - p_0(t)}{\Delta t} = -(\lambda_{01} + \lambda_{02}) p_0(t) + \lambda_{10}p_1(t) + \lambda_{20}p_2(t) + \frac{o(\Delta t)}{\Delta t} \tag{1}$$


II.
$$p_1(t) \lambda_{10} \Delta t + p_2(t) \lambda_{20} \Delta t$$



В соотношении $(1)$ перейдём к пределу, когда $\Delta t \to 0$.

$$\dot{p_0} (x) = - (\lambda_{01} + \lambda_{02}) p_0 (t) + \lambda_{10} p_1 (t) + \lambda_{20} p_2 (t)$$

Получим: - суммарный поток * вероятность + сумма попарного произведения.

$$\dot{p_1} (t) =  - (\lambda_{10} + \lambda_{13}) p_1 (t) + \lambda_{01} p_0 (t) + \lambda_{31} p_3 (t)$$
$$\dot{p_2} (t) = - (\lambda_{20} + \lambda_{23}) p_2 (t) + \lambda_{02} p_0 (t) + \lambda_{32} p_3 (t)$$
$$1 = p_0(t) + p_1(t) + p_2(t) + p_3(t)$$
